# Scene Checklist - 씬 체크리스트 (Active)

현재 작성 중인 씬에 대한 체크리스트. 매 씬 작성 전/후 확인.

## 기본 품질
- [ ] POV 일관성 유지됨
- [ ] 감각 묘사 최소 3개 포함
- [ ] 대화/행동/내면 균형
- [ ] 마크다운 형식 없음
- [ ] 캐릭터 대화 스타일 일관
- [ ] 씬 시작과 끝의 변화 존재
- [ ] 갈등/긴장 요소 존재

## 시대고증
- [ ] 의식주 시대 부합
- [ ] 관직/호칭 정확
- [ ] 무기/갑옷/전술 적합
- [ ] 현대 기술 부적절 사용 없음

## 무예/전투
- [ ] 기(氣) 단계 일관
- [ ] 전투 결과 합리적
- [ ] 주인공 전략/지략 우위 구도

## 바둑/전략
- [ ] 바둑 용어 정확한 대응
- [ ] 전략이 바둑적 사고 기반
- [ ] 수읽기/대국관 합리적 범위

## 독자 몰입
- [ ] Hook 존재 (1-5)
- [ ] Forward Pull 존재 (1-5)
- [ ] 감정적 공감 순간

## 서사 균형
- [ ] 줄기 반영 (천하/인간/귀로)
- [ ] 쾌감 패턴 배치
- [ ] 감각 재발견 (Part 1 특히)
