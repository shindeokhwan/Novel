# Main Plot - 메인 플롯

## 전체 구조
- **규모**: 150챕터, 4부작, ~450화 (150ch × 3scenes), ~495,000단어
- **서사 모델**: MICE-Character (1차) + Braid Structure + Rise Structure
- **핵심 질문**: 역사를 아는 자가 역사를 바꿀 수 있는가? 그리고 바꿔야 하는가?
- **바둑 대응**: Part 1-2 = 포석(布石), Part 3 = 중반전(中盤), Part 4 = 끝내기(ヨセ)
- **회귀 처리**: 회귀는 장르적 전제(출발점). 원인은 미스터리로 남김. AI 메커니즘 미공개. 순수 삼국지 통일에 집중.

## 주인공 성장 곡선
```
졸병 → 두목 → 현령 → 태수 → 주목 → 왕 → 통일자
 Ch.1   Ch.7   Ch.10   Ch.42  Ch.65  Ch.110  Ch.150
```

## 기(氣) 성장 곡선
```
무기 → 감기 → 운기 → [부상] → 발기 → 발기 한계(상한)
 Ch.1   Ch.9   Ch.22   Ch.45    Ch.58     Ch.150
```
