# Main Plot - 메인 플롯

## 전체 구조
- **규모**: 300챕터, 4부작, ~900화 (300ch × 3scenes), ~990,000단어
- **서사 모델**: MICE-Character (1차) + Braid Structure + Rise Structure
- **핵심 질문**: 역사를 아는 자가 역사를 바꿀 수 있는가? 그리고 바꿔야 하는가?
- **바둑 대응**: Part 1-2 = 포석(布石), Part 3 = 중반전(中盤), Part 4 = 끝내기(ヨセ)

## 주인공 성장 곡선
```
졸병 → 두목 → 현령 → 태수 → 주목 → 왕 → 통일자
 Ch.1   Ch.7   Ch.10   Ch.42  Ch.95  Ch.195  Ch.300
```

## 기(氣) 성장 곡선
```
무기 → 감기 → 운기 → [부상] → 발기 → 발기 한계(상한)
 Ch.1   Ch.9   Ch.22   Ch.45    Ch.72     Ch.300
```

- Ch.35 씬구성: (1) 전투 수습 — 포로 처우(만총 법치), 왕대 부상 / (2) 주방 답서 도착 — "때가 되면 다시 이야기합시다"(F042 진전) / (3) 서홍 혼자 사색 — "영천군의 포석은 완성되었다. 이제 예주다." 소아크 2-1 내적 결산. 귀로 줄기 짧게.
