# 캐릭터 프로필: 곽돌(郭突) — 악역 (Part 1)

## 기본 정보
- **이름**: 곽돌(郭突)
- **나이**: 30대 중반
- **성별**: 남
- **분류**: 창작 인물 (악역)
- **유형**: 무장+산적 두목
- **세력**: 황건적 잔당 → 독자 세력 (산적단)
- **악역 아크 범위**: Ch.8-14 (7챕터)
- **응징 챕터**: Ch.14

## 악역 설계

### 동기
- 원래 황건적 부장. 하곡촌 일대의 원래 주인이었으나 장삼(서진우)에게 밀려남.
- "내 땅을 빼앗은 놈" — 단순한 영토 탈환 동기.
- 부하들을 먹여살려야 하는 현실적 필요 + 체면/자존심.

### 악역 아크 4단계
1. **등장(Ch.8)**: 도적단 격퇴 직후 곽돌의 존재가 소문으로 전해짐. "서쪽에 큰 무리가 있다."
2. **에스컬레이션(Ch.11-13)**:
   - 겨울에 하곡촌 약탈 시도. 식량 일부 탈취.
   - **가상 캐릭터(곽철 또는 이만)에게 부상을 가함** — 직접적 피해.
   - 서진우의 외곽 초소 습격, 병사 사상.
3. **대결(Ch.13-14)**: 안평현 공방전. 곽돌이 병력 수에서 우세.
4. **응징(Ch.14)**: 서진우의 전술로 곽돌군 격파. **곽돌 직접 처단** → E9 카타르시스.

### 피해 목록
- 식량 탈취 (겨울 위기 심화)
- 병사 5-10명 전사
- 가상 캐릭터 1명 부상
- 외곽 초소 파괴

### 응징 방식
- 전술적 포위 → 항복 거부 → 전투 중 처단
- 서진우가 직접 또는 왕대/곽철이 처단 (주인공 주도)

## 무예/능력
- **기(氣) 단계**: 감기(感氣)~운기(運氣) 전조
- **전투 스타일**: 산적형. 기습과 약탈에 능하나 정면 대결에 약함.
- **병력**: ~300명 (잡병+황건 잔당)

## 성격
- 잔인하되 부하에게는 나름 의리가 있음 (순수 악 금지)
- 체면을 중시. 밀려난 것을 참지 못함.
- 지략은 부족. 힘으로 밀어붙이는 타입.
