# 씬 체크리스트 템플릿

## 기본 품질
- [ ] POV 일관성 유지됨 (씬 내 전환 없음)
- [ ] 감각 묘사 최소 3개 포함 (시각 편중 금지)
- [ ] 대화/행동/내면 균형 (35/40/25 가이드라인)
- [ ] 마크다운 형식 없음 (순수 산문)
- [ ] 캐릭터 대화 스타일 일관 (고어체 수준)
- [ ] 씬 시작과 끝의 변화 존재
- [ ] 갈등/긴장 요소 존재

## 시대고증
- [ ] 의식주가 후한 말~삼국시대에 부합
- [ ] 관직/작위 명칭 정확
- [ ] 무기/갑옷/전술 시대 적합
- [ ] 현대 기술/개념의 부적절한 사용 없음
- [ ] 인물 간 호칭이 관계/지위에 맞음

## 무예/전투
- [ ] 캐릭터 무예 수준이 기(氣) 단계와 일치
- [ ] 전투 결과가 전력 대비 합리적
- [ ] 주인공이 전략/지략으로 무예 부족 보완
- [ ] 전투 묘사가 감각적 (소리, 충격, 피)

## 바둑/전략
- [ ] 바둑 용어 사용 시 실제 개념과 정확히 대응
- [ ] 주인공의 전략이 바둑적 사고에 기반
- [ ] 수읽기/대국관 능력의 합리적 범위
- [ ] 전략 설명이 독자에게 이해 가능

## 독자 몰입
- [ ] Hook: 첫 문단에 몰입 유발 요소 (강도 1-5)
- [ ] Forward Pull: 씬 끝에 전진력 (강도 1-5)
- [ ] Mystery: 활성 미스터리 1개 이상 언급/힌트
- [ ] Emotional Anchor: 감정적 공감 순간 존재

## 서사 균형
- [ ] Strand: 계획한 줄기 (천하/인간/귀로) 반영
- [ ] Entertainment: 쾌감 패턴 계획대로 배치
- [ ] 감각 재발견: (Part 1 특히) 전신마비→건강체 감격 묘사
