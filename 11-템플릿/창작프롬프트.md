# 창작 프롬프트 가이드 - Thriple Kingdom

## 씬 작성 프롬프트

### 전투 씬
```
[챕터/씬 번호]의 전투 장면을 작성해줘.
- 장소: [전장 위치]
- 아군: [병력/장수]
- 적군: [병력/장수]
- 주인공의 전략: [바둑 개념 기반 전략]
- 전투 결과: [승/패/무승부]
- 핵심 감정: [감정]
- 감각 묘사 포인트: [소리/시각/촉각]
```

### 전략 회의 씬
```
[챕터/씬 번호]의 전략 회의를 작성해줘.
- 장소: [회의 장소]
- 참석자: [인물 목록]
- 안건: [논의 주제]
- 주인공의 바둑적 분석: [포석/수읽기/대국관 적용]
- 갈등: [의견 대립 포인트]
- 결론: [결정 사항]
```

### 감각 재발견 씬
```
[챕터/씬 번호]에서 주인공의 감각 재발견을 작성해줘.
- 상황: [어떤 맥락에서]
- 재발견하는 감각: [시각/청각/촉각/후각/미각]
- 현대 기억 대비: [전신 마비 시절과의 대조]
- 감정: [감격/놀라움/그리움]
```

### 인재 영입 씬
```
[챕터/씬 번호]의 인재 영입 장면을 작성해줘.
- 대상: [영입할 인물]
- 주인공의 역사 지식: [이 인물에 대해 아는 것]
- 접근 전략: [바둑 포석 개념 적용]
- 상대의 저항/조건: [왜 쉽지 않은지]
- 설득 포인트: [핵심 설득 논리]
```

## 캐릭터 관련 프롬프트

### 실존 인물 생성
```
[인물명]의 캐릭터 프로필을 생성해줘.
- 역사적 기록 기반 + 소설적 해석
- 주인공과의 관계 설정
- 무예/지략 수준
- 주인공이 역사 지식으로 아는 이 인물의 미래
```

### 감정 분석
```
[인물명]의 Ch.[N] 씬 [N] 감정을 분석해줘.
- 직전 사건: [무슨 일이 있었는지]
- 8감정 스코어링
- 3층 분석 (표면/의식/무의식)
- 회귀 심리 (주인공의 경우)
```

## 구조 관련 프롬프트

### 챕터 아웃라인
```
Ch.[N] "[제목]"의 아웃라인을 작성해줘.
- Part: [Part N]
- 시기: [년, 월]
- 성장 단계: [현재 단계]
- 핵심 사건: [이 챕터의 핵심]
- 바둑 대응: [사용할 바둑 개념]
- 쾌감 패턴: [E1-E8 중]
- 서사 줄기: [천하/인간/귀로 비율]
```

### 바둑-전략 매핑
```
[전략/전투]를 바둑 개념으로 매핑해줘.
- 현재 상황을 바둑판으로 표현
- 주인공의 다음 수
- 적의 예상 대응
- 최선의 수순
```
