## 1. 시대착오어 절대 금지 (11종)

삼국시대(2세기 말~3세기) 배경. 대사·서술·내면 독백 막론하고 아래 단어 사용 절대 금지.

| 금지 단어 | 대체 표현 |
|----------|----------|
| 타이밍 | 시기(時機) / 때 / 순간 |
| 리듬 | 박자(拍子) / 흐름 |
| 패턴 | 수순(手順) / 습관 / 결 |
| 시스템 | 체계(體系) / 제도(制度) |
| 스타일 | 방식(方式) / 방법(方法) |
| 카리스마 | 위엄(威嚴) / 덕망(德望) |
| 퍼센트(%) | 완전히 / 온전히 / 전부 |
| 이미지 | 모습 / 형상(形象) |
| 레벨 | 수준 / 단계 |
| 에너지 | 기력(氣力) / 힘 |
| 포인트 | 요점 / 핵심 |

## 2. POV 규칙

- 3인칭 제한 시점, 과거형
- 씬 내 POV 전환 절대 금지
- POV 캐릭터의 감각/생각만 직접 서술
- 다른 캐릭터의 내면은 추측/관찰로만

## 3. 산문 형식

- 원고 내 마크다운 형식 절대 금지 (헤더, 불릿, 표, 볼드 등)
- 순수 산문, 단락은 빈 줄로 구분
- 대화는 큰따옴표, 씬 전환은 "* * *"
- 챕터 번호 산문 참조 금지 ("Ch.X에서" → 시간/장소/사건으로)

## 4. 문장 단편화 금지

- SVO 단문 3문장 이상 연속 금지 → 접속어미(~면서/~며/~한 뒤)로 통합
- 상세 규칙 SF01-SF05: `reference/style-guide.md` 문장 단편화 방지 규칙 참조

## 5. 대화 패턴 금지

- **분리호칭 절대 금지**: `"조우." 서홍이 말했다. "내용"` → 합치기 또는 행동 비트
- **`"[대사]." [이름]이/가 말했다.` 단독 금지**: 수식어·행동 비트 없이 대사+말했다만 = 금지
- 대안: 태그 삭제 / 행동 비트 / 답했다·받았다·끼어들었다·덧붙였다

## 6. em-dash(—) 상한

- 챕터당 최대 5회, 같은 문단 내 2회 금지. 상세: `reference/style-guide.md` 구두점 규칙

## 7. 기본 서술 원칙

- Show Don't Tell: 감정 직접 명명 금지 → 신체 반응/행동으로
- 감각 묘사: 씬당 최소 3감각 / 대화 노이즈: 챕터당 최소 3회
- 수치 시간("0.1초") 금지 → 체감적 표현

## 8. 분량 기준

- 챕터 전체: 9,000~12,000자, 최소 7,000자
- 씬 1개: 3,000~4,000자, 최소 2,000자

## 9. 대화 진정성 (from humanization-rules)

- 정보전달형 대사 금지: 상대가 이미 아는 정보를 독자를 위해 설명하는 대사 불가
- 감정 라벨링 대사 금지: "나 화났어" 식 직접 명명 → Show don't tell
- **대화 노이즈 5종** (챕터당 최소 3회): 끊김 / 딴소리 / 비완성 문장 / 오해 / 침묵
- 캐릭터 음성 구분: 대사만 읽어도 화자 식별 가능해야 함

## 10. 갈등 깊이 + 비합리성 (from humanization-rules)

- 갈등 지속 최소: 소갈등 2ep / 중갈등 4ep / 대갈등 8ep
- 해결 후 잔여 긴장 의무: 최소 2ep 동안 미세한 어색함·달라진 거리감 유지
- 해결 금지: 한 번의 대화로 완전 해소 / 모든 관계자 동시 납득 / 해결 후 즉시 이전보다 나은 관계
- **비합리성 쿼터**: 챕터당 최소 1회 — 감정에 의한 비논리적 행동, 자기기만, 회피, 과잉반응 중 택1
- 인간적 비효율: 의미 없이 검 뽑았다 넣기, 이미 확인한 지도 다시 펼치기 등

## 11. 에피소드 종결 5종 순환 (from humanization-rules)

끊김형 / 대화잔향형 / 감각단독형 / 질문형 / 타인시선형 — 최소 5가지 순환.
- 직전 챕터와 동일 엔딩 방식 **절대 금지**
- 감각단독형·질문형 연속 3회 금지 / 끊김형·대화잔향형 10ch당 각 1회 이상

## 12. 쾌감 패턴 8종 (from entertainment-density)

| ID | 패턴 | ID | 패턴 |
|----|------|----|------|
| E1 | 역전승 | E5 | 감정 폭발 |
| E2 | 오해 해소 | E6 | 결단 |
| E3 | 숨겨진 능력 발현 | E7 | 진실 발각 |
| E4 | 복선 회수 | E8 | 규모 확장 |

- 빈도: 에피소드당 미니1+, 챕터당 중형1-2, 대형0-1
- 동일 E패턴 **연속 2회 금지** (챕터 연속 기준)
- 축적 3-5ep → 해소 1-2ep 순환. 10ep 이상 해소 없으면 위험, 3ep 연속 대형은 인플레
- **안티패턴**: 쾌감 인플레 / 공허한 쾌감(축적 없는 해소) / 먹물주인공 / 역사 파괴 쾌감 / 무예 인플레
- 불확실성 보존: 5ch당 1회 — 성공+부작용 / 실패+이득 / 중단 종결 중 택1

## 13. 챕터 성격 5유형 + 클라이맥스 배치 (from plot-structure)

| 유형 | 정의 | 씬 구성 |
|------|------|--------|
| 포석형 | 정보 수집, 계획, 관계 형성 | 설정→심화→불확실성 잔존 |
| 돌파형 | 직접 행동, 전투, 결단 | 긴장→폭발→여진 |
| 여운형 | 직전 사건 처리, 감정 정리 | 상태 확인→해석→새 의문 발아 |
| 인간형 | 전략 없이 사람과 사람의 이야기 | 일상→균열→온기 |
| 전환형 | 기존 계획을 뒤집는 사건 | 예상→파열→재계산 |

- 같은 유형 **3회 연속 금지** / 인간형 5ch당 1회 이상 / 전환형 소아크당 2회 이상

**씬 클라이맥스 배치**:
- 전면배치: 씬1=사건→씬2=파장→씬3=새 질문
- 중간배치: 씬1=축적→씬2=폭발→씬3=여운
- 후면배치: 씬1=준비→씬2=심화→씬3=카타르시스
- 후면배치만 연속 4회 금지 / 5ch당 전면1회+중간1회 이상 필수
