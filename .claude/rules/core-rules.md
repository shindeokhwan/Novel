## 1. 시대착오어 절대 금지 (11종)

삼국시대(2세기 말~3세기) 배경. 대사·서술·내면 독백 막론하고 아래 단어 사용 절대 금지.

| 금지 단어 | 대체 표현 |
|----------|----------|
| 타이밍 | 시기(時機) / 때 / 순간 |
| 리듬 | 박자(拍子) / 흐름 |
| 패턴 | 수순(手順) / 습관 / 결 |
| 시스템 | 체계(體系) / 제도(制度) |
| 스타일 | 방식(方式) / 방법(方法) |
| 카리스마 | 위엄(威嚴) / 덕망(德望) |
| 퍼센트(%) | 완전히 / 온전히 / 전부 |
| 이미지 | 모습 / 형상(形象) |
| 레벨 | 수준 / 단계 |
| 에너지 | 기력(氣力) / 힘 |

## 2. POV 규칙

- 3인칭 제한 시점, 과거형
- 씬 내 POV 전환 절대 금지
- POV 캐릭터의 감각/생각만 직접 서술
- 다른 캐릭터의 내면은 추측/관찰로만

## 3. 산문 형식

- 원고 내 마크다운 형식 절대 금지 (헤더, 불릿, 표, 볼드 등)
- 순수 산문, 단락은 빈 줄로 구분
- 대화는 큰따옴표, 씬 전환은 "* * *"
- 챕터 번호 산문 참조 금지 ("Ch.X에서" → 시간/장소/사건으로)

## 4. 문장 단편화 금지

- SVO 단문 3문장 이상 연속 금지 → 접속어미(~면서/~며/~한 뒤)로 통합
- **정보 쪼개기 금지**: 한 문장으로 합칠 수 있는 동작+대상+속성을 여러 문장으로 분절하지 말 것. (예: "진무가 들어왔다. 주민 대표를 데리고 있었다. 노인이었다." → "진무가 주민 대표를 데리고 들어왔다. ~한 노인이었다.")
- **주어 중복 재소개 금지**: 한 문장에서 소개한 주어를 바로 다음 문장에서 다시 소개하지 말 것. (예: "상인이 전했다. 남쪽에서 올라온 상인이었다." → "남쪽에서 올라온 상인이 전한 것이었다."로 합침)
- **어미 단조 반복 금지**: 같은 어미(~했다/~했다고 했다/~이었다)가 3문장 이상 연속 금지. 어미를 변형(~다고/~라고/접속어미)하거나 문장 구조를 바꿀 것.
- **시나리오식 컷 나열 금지**: 한 줄짜리 짧은 문장을 독립 단락으로 연속 배치하는 것 금지. (예: "문이 열렸다." / "앉았다." / "달랐다." — 각각 한 줄 단락) 소설은 영화 시나리오가 아니다. 행동·감각·판단을 하나의 단락 흐름 안에서 연결할 것. 단, 극적 강조를 위한 의도적 단문 1회는 허용하되 연속 사용 금지.
- 상세 규칙 SF01-SF05: `reference/style-guide.md` 문장 단편화 방지 규칙 참조

## 5. 대화 패턴 금지

- **분리호칭 절대 금지**: `"조우." 서홍이 말했다. "내용"` → 합치기 또는 행동 비트
- **`"[대사]." [이름]이/가 말했다.` 단독 금지**: 수식어·행동 비트 없이 대사+말했다만 = 금지
- **동일 화자 대사 분절 금지**: 같은 화자의 연속 발화를 `"대사1." 서술. "대사2."` 형태로 쪼개는 것 금지. AI 흔적 패턴.
  - **금지**: `"감이 끊겼지." 곽철이 고개를 저었다. "아까는 됐는데 지금은 안 된 거잖아."`
  - **대안1 (대사 합치기)**: `곽철이 고개를 저었다. "감이 끊겼지. 아까는 됐는데 지금은 안 된 거잖아."`
  - **대안2 (행동 후치)**: `"감이 끊겼지. 아까는 됐는데 지금은 안 된 거잖아." 곽철이 고개를 저었다.`
  - **예외**: 대사 사이에 실제 시간적 간격(침묵 3초+, 상대 반응, 행동 변화)이 있을 때만 분리 허용
- 대안: 태그 삭제 / 행동 비트 / 답했다·받았다·끼어들었다·덧붙였다

## 6. em-dash(—) 상한

- 챕터당 최대 5회, 같은 문단 내 2회 금지. 상세: `reference/style-guide.md` 구두점 규칙

## 7. 기본 서술 원칙

- Show Don't Tell: 감정 직접 명명 금지 → 신체 반응/행동으로
- 감각 묘사: 씬당 최소 3감각 / 대화 노이즈: 챕터당 최소 3회
- 수치 시간("0.1초") 금지 → 체감적 표현

## 8. 분량 기준

- 챕터 전체: 9,000~12,000자, 최소 7,000자
- 씬 1개: 3,000~4,000자, 최소 2,000자

## 9. 대화 진정성 (from humanization-rules)

- 정보전달형 대사 금지: 상대가 이미 아는 정보를 독자를 위해 설명하는 대사 불가
- 감정 라벨링 대사 금지: "나 화났어" 식 직접 명명 → Show don't tell
- **서술형 대사 금지**: 대사가 서술/요약처럼 읽히면 안 된다. 캐릭터가 아니라 작가가 상황을 설명하는 것처럼 들리는 대사 금지. (예: "내일 주민 의논에서 현을 맡기겠소" → "원노인이 현을 내줄 거요") 대사는 그 캐릭터가 실제로 입에 올릴 말이어야 하며, 정보 압축·플롯 요약·무대지시 역할을 해서는 안 된다.
- **리액션 중복 금지**: 한 비트에 캐릭터 반응 동작을 2개 이상 나열 금지. "눈이 좁아졌다. 고개를 갸웃했다." → 하나만 선택.
- **에코 대사 금지**: 상대 단어를 그대로 되받아 의미 부여하는 패턴 금지. ("일단은." / "일단은." 식 반복) 되받으려면 해석·반론·감정을 실어야 함.
- **캐릭터 음량 일관**: 과묵한 캐릭터가 갑자기 3~4문장 설명, 수다스러운 캐릭터가 갑자기 한마디 — 캐릭터 프로필의 대화 분량을 벗어나는 음량 이탈 금지. 정보량이 많으면 대화를 쪼개거나 서술로 보충.
- **대화 노이즈 5종** (챕터당 최소 3회): 끊김 / 딴소리 / 비완성 문장 / 오해 / 침묵
- 캐릭터 음성 구분: 대사만 읽어도 화자 식별 가능해야 함

## 10. 갈등 깊이 + 비합리성 (from humanization-rules)

- 갈등 지속 최소: 소갈등 2ep / 중갈등 4ep / 대갈등 8ep
- 해결 후 잔여 긴장 의무: 최소 2ep 동안 미세한 어색함·달라진 거리감 유지
- 해결 금지: 한 번의 대화로 완전 해소 / 모든 관계자 동시 납득 / 해결 후 즉시 이전보다 나은 관계
- **비합리성 쿼터**: 챕터당 최소 1회 — 감정에 의한 비논리적 행동, 자기기만, 회피, 과잉반응 중 택1
- 인간적 비효율: 의미 없이 검 뽑았다 넣기, 이미 확인한 지도 다시 펼치기 등

## 11. 에피소드 종결 5종 순환 (from humanization-rules)

끊김형 / 대화잔향형 / 감각단독형 / 질문형 / 타인시선형 — 최소 5가지 순환.
- 직전 챕터와 동일 엔딩 방식 **절대 금지**
- 감각단독형·질문형 연속 3회 금지 / 끊김형·대화잔향형 10ch당 각 1회 이상

## 12. 쾌감 패턴 8종 (from entertainment-density)

| ID | 패턴 | ID | 패턴 |
|----|------|----|------|
| E1 | 역전승 | E5 | 감정 폭발 |
| E2 | 오해 해소 | E6 | 결단 |
| E3 | 숨겨진 능력 발현 | E7 | 진실 발각 |
| E4 | 복선 회수 | E8 | 규모 확장 |
| E9 | 악역 응징 | | |

- 빈도: 에피소드당 미니1+, 챕터당 중형1-2, 대형0-1
- 동일 E패턴 **연속 2회 금지** (챕터 연속 기준)
- 축적 3-5ep → 해소 1-2ep 순환. 10ep 이상 해소 없으면 위험, 3ep 연속 대형은 인플레
- **안티패턴**: 쾌감 인플레 / 공허한 쾌감(축적 없는 해소) / 먹물주인공 / 역사 파괴 쾌감 / 무예 인플레
- 불확실성 보존: 5ch당 1회 — 성공+부작용 / 실패+이득 / 중단 종결 중 택1

## 13. 챕터 성격 5유형 + 클라이맥스 배치 (from plot-structure)

| 유형 | 정의 | 씬 구성 |
|------|------|--------|
| 포석형 | 정보 수집, 계획, 관계 형성 | 설정→심화→불확실성 잔존 |
| 돌파형 | 직접 행동, 전투, 결단 | 긴장→폭발→여진 |
| 여운형 | 직전 사건 처리, 감정 정리 | 상태 확인→해석→새 의문 발아 |
| 인간형 | 전략 없이 사람과 사람의 이야기 | 일상→균열→온기 |
| 전환형 | 기존 계획을 뒤집는 사건 | 예상→파열→재계산 |

- 같은 유형 **3회 연속 금지** / 인간형 5ch당 1회 이상 / 전환형 소아크당 2회 이상

**씬 클라이맥스 배치**:
- 전면배치: 씬1=사건→씬2=파장→씬3=새 질문
- 중간배치: 씬1=축적→씬2=폭발→씬3=여운
- 후면배치: 씬1=준비→씬2=심화→씬3=카타르시스
- 후면배치만 연속 4회 금지 / 5ch당 전면1회+중간1회 이상 필수

## 14. 전투 비중 (from style-guide-battle)

- **전체 40% 이상 전투 관련 챕터** 목표
- 부대전이 전투 챕터의 **주(60%+)**, 개인전은 하이라이트(40%-)
- 부대전 필수 시퀀스: 배치→접촉→밀고당김→위기→전환점→결말
- 10챕터당 최소 1회 **타 진영 POV 전투씬** (조조/원소/유비/손권 시점)
- 전투 결과 분배: 먹치킨 30% / 대등 40% / 기지 20% / 패배 10%
- 10챕터당 최소 1회 주인공 측 **좌절/패배**

## 15. 대화 흐름 (from style-guide)

- 활발한 대화 중 서술 삽입 **최대 2문장**
- 허용 조건: 의미 있는 침묵, 장면 변화 행동, 내적 깨달음
- 금지: `"대사" + 3-5줄 서술 + "대사" + 3-5줄 서술` 반복 패턴
- 대화 시퀀스: 연속 대화(5-10줄) → 정리 단락 → 다음 대화
- **대화 구체성 필수**: 대사는 항상 구체적 정보를 담아야 함. 추상적·압축적 핑퐁("어떠했소" / "생각해보겠다고 했어" / "먼저였소" / "알아") 금지. 보고·분석·판단 대화에서는 면담 내용, 상대 발언, 조건 등 실제 정보를 언급해야 함. 독자가 대화만 읽어도 상황을 파악할 수 있어야 한다.

## 16. 악역 카타르시스 (from character-management)

- 매 소아크당 **최소 1명 네임드 악역**
- 악역 아크: 등장(위협) → 에스컬레이션(피해) → 대결 → **응징(카타르시스)**
- 악역이 주인공/부하에게 **직접적 피해** 필수
- 악역 최소 지속: **5챕터 이상**
- 응징 없는 악역 퇴장 금지

## 17. 주인공 감정 성장 (from character-management)

- Part 1: 어리바리, 적응 실패, 주변이 구해줌
- Part 2 전반: 자리 잡기, 어색한 농담 시도
- Part 2 후반: 진정한 온기, 자연스러운 유머
- Part 3: 공개적 챙김, 자연스러운 농담, 결정의 무게
- Part 4: 완성된 인격체, 따뜻하되 결단력

## 18. 조연 지속 등장 (from character-management)

## 19. 선행맥락 규칙

모든 서술 비트는 독자가 납득할 선행 맥락이 있어야 한다. 맥락 없이 등장하는 서술은 뜬금없음을 유발한다.

- **대사**: 대화적 촉발(상대 발언·상황 변화)이 선행해야 함. 아무 맥락 없는 독백/발화 금지.
- **캐릭터 행동**(합류·이탈·잔류): 명시적 동기가 선행해야 함. 자연스러운 합류라도 "왜 계속 함께하는가"에 답이 있어야 함.
- **사건 참조**: 참조하는 사건이 실제로 서술되어 있어야 함. 일어나지 않은 장면을 본 것처럼 쓰는 것 금지.
- **비교·평가**: 기준점이 먼저 제시되어야 함. 비교 대상의 수준이 독자에게 전달된 뒤에 비교.

## 20. 캐릭터 정보 경계 (작가 지식 ≠ 캐릭터 지식)

캐릭터는 작중에서 직접 보거나, 듣거나, 추론할 근거가 있는 정보만 알 수 있다. 작가가 설정으로 알고 있는 정보를 캐릭터가 앞서 아는 것은 금지.

- **질문·발언**: 캐릭터가 상대의 신분·배경을 전제하는 질문/발언은, 그 전제를 알게 된 근거가 작중에 있어야 함. 근거 없이 단정하는 질문 금지. (예: 군인인지 모르면서 "어디 군이었어?" → "전에 뭘 했어?")
- **반응·판단**: 캐릭터가 아직 모르는 정보에 기반한 감정 반응·전략 판단 금지.
- **내면 서술**: POV 캐릭터의 내면에서 작가만 아는 설정을 기정사실로 서술 금지. 추측이면 추측으로 써야 함.

## 21. 대사 존비어 일관성

한 캐릭터의 대사 안에서 존대와 반말이 섞이면 안 된다. 한 발화 단위 내에서 어체를 통일할 것.

- **존대 문장 + 반말 호칭 금지**: "~했소"체에 "너희" 금지 → "당신들/그대들"로 통일. (예: "너희가 오는 것을 봤소" → "당신들이 오는 것을 봤소")
- **반말 문장 + 존대 호칭 금지**: "~했어"체에 "어르신" 등 과도한 존칭 혼용 주의.
- **관계 변화 시 전환은 허용**: 초면→친밀로 어체가 바뀌는 것은 가능하되, 전환 시점이 명확해야 함. 같은 대화 블록 안에서 왔다 갔다 금지.
- **호칭은 지위 변경 시점에 맞춰 전환**: 두목·현령 등 직위가 부여된 이후, 아군 캐릭터는 새 호칭을 사용해야 함. 이전 이름(장삼 등)으로 부르는 것은 적대 세력·외부인·의도적 전환 장면에만 허용. (예: 두목 취임 후 진무가 "장삼" → "두목"으로 전환)

## 22. 능력 성장 점진성

능력(감기·무예·지략 등)이 새로운 수준에 도달하려면 **3단계**를 거쳐야 한다. 단계를 건너뛰는 급성장 금지.

1. **한계 인식**: 현재 수준의 한계를 작중에서 명시적으로 보여줄 것. (예: "감기가 수십 보 밖에 닿지 않았다")
2. **수련/계기**: 한계를 넘는 훈련·전투·깨달음 장면이 선행해야 함. 이전 챕터 또는 같은 챕터 내에서 성장의 과정을 보여줄 것.
3. **확장 확인 + 이전 대비**: 새 수준을 보여줄 때, 이전 한계를 대비시켜 성장을 체감하게 할 것. (예: "첫 습격 때는 닿지 않았다. 지금은 달랐다")

- 능력 점프가 2챕터 이상 떨어져 있으면 반드시 중간에 수련/성장 비트를 삽입
- 독자가 "언제 이렇게 강해졌지?"라고 느끼면 실패

## 23. 전투 종결의 물리적 인과

전투의 승패는 **부상 축적 → 전투력 저하 → 물리적 패배**의 인과 사슬로 결정되어야 한다.

- **부상 축적**: 전투 중 부상은 단계적으로 쌓여야 함. 경미한 상처 하나로 패배 금지. (예: 옆구리 스침 → 팔뚝 근육 절단 → 허벅지 → 도끼 탈락)
- **부상 연속성**: 한 장면에서 입은 부상은 이후 장면에서 반드시 반영. 중간에 부상이 리셋되거나 무시되는 것 금지.
- **종결 무게 비례**: 악역/대결의 종결은 축적된 서사 분량에 비례하는 무게를 가져야 함. 5챕터+ 악역이 3줄로 끝나면 실패.
- **물리적 근거**: 승리에는 명확한 물리적 원인이 있어야 함. "한 번 쳐내서 끝" 금지. 체력 소모, 부상 누적, 전술적 궁지 중 하나 이상이 패배의 원인으로 작동해야 함.
- **무기 상실 = 즉시 패배 금지**: 무기를 떨어뜨리는 것만으로 전투 종결 금지. 무기 상실은 부상 축적의 결과여야 함.
