# Character Management Rules - 캐릭터 관리 규칙

캐릭터 파일 구조, 감정 추적 시스템, 아크 추적, 대화 스타일 관리 규칙을 정의한다.

```

├── 조연/
│   ├── [이름].md
│   └── [이름]_감정추적.md
└── 엑스트라/
    └── [그룹명].md            # 그룹 단위 관리
```

## Character Profile Requirements

### 필수 항목
- 기본 정보 (이름, 자(字), 나이, 성별, 역할, 첫 등장)
- 분류 (실존 인물 / 창작 인물)
- 외모 (키/체형, 머리, 눈, 특징, 복장, 무기)
- 배경 (출신, 가문, 관직, 세력, 사회적 위치)
- 성격 (핵심 특성 3개, 강점, 약점, 두려움)
- 욕구 (의식적 목표, 무의식적 필요)
- 무예/능력 (기(氣) 단계, 무기 특기, 특수 능력, 전략/지략 수준)
- 대화 스타일 (어휘 수준, 말투, 고유 표현, 금기어)
- 캐릭터 아크 (시작점, 전환점, 도착점)

### 삼국지 특화 항목
- 인물 유형: 무장 / 모사(참모) / 군주 / 간웅 / 상인 / 의사 / 기타
- 세력 소속 변화 이력
- 관직 변화 이력
- 역사적 최후 (실존 인물의 경우)
- 주인공과의 관계 변화 궤적

### 주인공 전용 항목
- 현대 기억 (2030년대 경험, 바둑 경력, 전신 마비 경험)
- 바둑적 사고 패턴 (수읽기, 대국관, 포석 능력 수준)
- 감각 재발견 로그 (어떤 감각을 언제 재발견했는지)
- 역사 지식 수준 (삼국지연의 기준, 정사와의 차이)

## Emotion Tracking System

### 8가지 기본 감정 (1-10 척도)
기쁨, 두려움, 분노, 슬픔, 혐오, 놀라움, 신뢰, 기대

### 기록 시점
- 매 씬 종료 후 주요 감정 변화 기록
- 중대한 사건(전투, 배신, 동맹) 발생 시 즉시 기록
- 캐릭터 아크 전환점에서 상세 기록

### 감정 기록 형식
```
## 챕터 X, 씬 Y
- 지배 감정: [감정](점수)
- 보조 감정: [감정](점수)
- 트리거: [감정 변화 원인]
- 신체 반응: [표현된 신체 반응]
- 이전 대비 변화: [증가/감소/유지]
- 바둑 심리: [승부사 감각 발동 여부]
```

## Dialogue Style Rules

### 작성 전 확인
1. 캐릭터 파일의 대화 스타일 섹션 읽기
2. 현재 감정 상태 확인
3. 대화 상대와의 관계 역학 (상하관계, 적아) 파악

### 시대적 대화 원칙
- 기본: 고어체 기반의 현대적 가독성 (번역체 느낌)
- 주인공 내면: 현대적 사고 (바둑 용어, 현대 비유)
- 주인공 대화: 점차 시대에 적응 (초반 어색 → 후반 자연스러움)
- 무장: 호쾌하고 직설적
- 모사: 우회적이고 비유적
- 군주: 위엄 있으나 인간적

### 금지 사항
- 캐릭터 간 말투 혼동
- 감정 상태와 무관한 말투
- 캐릭터가 모르는 정보 언급
- 현대 속어/인터넷 용어 (주인공 내면 제외)
