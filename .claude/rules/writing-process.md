# Writing Process Rules - 집필 워크플로우 규칙

삼국지 회귀물 소설 작성의 전/중/후 프로세스와 파일 업데이트 트리거를 정의한다.

## Before Writing (작성 전 - 필수)

1. `00-핵심관리/현재작업상태.md` 읽기
2. 관련 캐릭터 파일 확인 (`02-캐릭터/`)
3. 현재 챕터 컨텍스트 파악
4. `03-플롯/플롯홀.md`, `03-플롯/복선추적.md`, `03-플롯/인물관계도.md` 검토
5. 타임라인 일관성 검증 (`01-세계관/타임라인.md`)
6. 역사적 맥락 확인 (`01-세계관/역사배경.md`)
7. 캐릭터 위치, 세력 배치, 군사력 검증
8. `03-플롯/서사줄기추적.md` 최근 5챕터 균형 확인 → 이번 씬 줄기 결정
9. 이전 2-3 에피소드 원고 읽기 (문맥 연속성 및 톤 확보)
10. 바둑-전략 대응 현황 확인 (현재 포석/중반전/끝내기 단계)
11. **능력 업그레이드 빈도 점검** (`ability-upgrade.md`) — 직전 소(小) 업그레이드 이후 챕터 수 확인. 4챕터 초과 시 이번 챕터에 소 업그레이드 필수 배치

## 챕터 분량 기준 (Chapter Length Standard) — 필수 준수

### 분량 목표 (한글 글자 수 기준)
| 단위 | 목표 | 최솟값 | 비고 |
|------|------|--------|------|
| 챕터 전체 | 9,000~12,000자 | **7,000자** | 미달 시 즉시 씬 확장 |
| 씬 1개 | 3,000~4,000자 | 2,000자 | 3씬 구성 기본 |

**기준 근거**: Ch.001~015 평균 9,097자. 1화(12,931자), 2화(9,222자)가 기준치.

### 예외 케이스
- **클라이맥스 챕터** (소아크 마지막, 대전투): 12,000자 이상 허용
- **전환/연결 챕터**: 최소 5,500자 (이하 금지)

### 분량 확장 방법 (단순 패딩 금지 — 밀도 있는 확장만)
1. **내면 묘사 확장**: 서홍의 판단 근거·고뇌·수읽기 과정을 최소 3문단 이상 풀어쓰기
2. **바둑 전략 상세화**: 바둑 개념 → 군사/정치 적용 과정을 독자가 이해하도록 서술
3. **대화 비트 확장**: 대화 전후 행동·신체 반응·내면 반응을 1~3문장 추가
4. **씬 전환 여운**: 현재 씬의 결과·감정·분위기를 정리하는 단락 추가
5. **감각 묘사 충실화**: 시각 외 청각·촉각·후각·미각을 충분한 문장으로 서술

### 분량 점검 (퇴고 체크리스트 추가 항목)
- [ ] 챕터 총 글자 수 확인 → 7,000자 미만 시 확장 필수
- [ ] 씬별 글자 수 확인 → 2,000자 미만 씬 존재 시 확장

## During Writing (작성 중 - 준수)

1. 씬 내 POV 일관성 유지 (전환 금지)
2. 캐릭터별 말투 적용 (캐릭터 파일의 대화 스타일)
3. 오감 활용 감각적 묘사 (최소 3개 감각)
4. 대화, 행동, 내면 묘사 균형
5. 시대고증 준수 (의식주, 관직, 무기, 문화)
6. 바둑 용어의 자연스러운 전략 적용
7. 마크다운 형식 사용 금지 (원고 내)
8. 자연스러운 산문 형식 유지
9. 전신 마비→건강체 감각 재발견 묘사 (초반부 특히 중요)
10. `03-플롯/씬체크리스트.md` 확인
11. **능력 업그레이드 에피소드 배치** — `ability-upgrade.md` 레퍼토리(W01-W07, I01-I06, G01-G06, L01-L05) 참조, 해당 챕터에 적합한 소 업그레이드 씬 내 통합
12. **전투씬 서술 원칙 준수** — 서홍 참여 전투 시 개인 전투 5교환 이상 묘사, 수읽기 내면 독백 포함, 미세 먼치킨 적용. `style-guide.md` 전투씬 서술 원칙 참조
13. **대인/집단 전투 원칙 준수** — 장수급 대결 시: 이름 부여 + 이름 교환 대사 + 치명적 결말 + 위력 과시 주변 반응 포함. 집단전 시: 승리 논리 명시 + 개인전-집단전 교차 편집. `style-guide.md` 대인/집단 전투 원칙 참조

## After Writing (작성 후 - 필수)

1. `00-핵심관리/진행상황.md` 업데이트
2. 감정 추적 기록 (`05-심리추적/감정추적/`)
3. `01-세계관/타임라인.md` 업데이트
4. 새로운 복선, 캐릭터 발전, 세계관 세부사항 기록
5. 플롯 일관성, 캐릭터 목소리, 페이싱 최종 확인
6. `03-플롯/서사줄기추적.md`에 이번 씬/챕터 줄기 비율 기록
7. 역사 변경 사항 기록 (원래 역사 vs 변경된 역사)
8. 무예/내공 수준 변화 기록
9. **인간화 리뷰 실행 (필수)** — `.claude/skills/SKILL.md` 워크플로우에 따라 작성된 원고의 AI 흔적을 검출하고 인간화 점수를 산출한다. 점수 70 미만 시 해당 부분 즉시 수정 후 재검사. 리포트는 `08-피드백/인간화리뷰_[날짜].md`에 저장.
10. **퇴고 실행 (필수, 예외 없음)** — 초고 완성 즉시 1차수정(퇴고)을 진행한다. 별도 지시 없이도 자동으로 수행. 퇴고 완료 후 `ch-[NNN]_v2_1차수정.md` 생성.
11. **능력 업그레이드 기록** — `ability-upgrade.md` 기록 형식에 따라 트랙/규모/레퍼토리 코드/4요소 체크 기록. 사용된 쾌감 패턴(E3 등) 함께 기재.

### 퇴고 체크리스트 (1차수정 필수 항목)
- [ ] em-dash(—) 챕터당 5회 이하 확인, 4조건 미충족 시 대체
- [ ] 메타 서술 라벨 제거 ("수읽기가 시작됐다", "플래시백" 등 선언형)
- [ ] 독자 설명 라벨 제거 (독자를 위해 캐릭터 행동을 설명하는 문장)
- [ ] 중복 설명 제거 (앞 문장이 이미 전달한 내용을 다음 문장에서 반복)
- [ ] 에피소드 종결 패턴 — 직전 챕터와 동일한 방식 금지
- [ ] 인간화 규칙 위반 패턴 확인 (수치 시간, 동일 신체 표현 반복 등)
- [ ] POV 일관성 씬 내 유지 여부
- [ ] 대화 노이즈 최소 3회 (끊김/딴소리/비완성 문장) 확인
- [ ] 능력 업그레이드 에피소드 점검 — 직전 4챕터 내 소 업그레이드 0회면 이번 챕터에 추가 필수. 배치 후 `ability-upgrade.md` 기록 형식으로 기재
- [ ] **대인 전투 퇴고** — 장수급 적에게 이름 부여됐는가? 이름 교환 대사 있는가? 결말이 치명적 타격으로 종결됐는가? 위력 과시 후 주변 반응 묘사 있는가?
- [ ] **집단 전투 퇴고** — 수적 열세 승리 시 전술적 이유 명시됐는가? 개인전-집단전 교차 편집 구성됐는가? 포위 효과 서술 있는가?
- [ ] **문장 단편화 점검** — 행동 묘사 구간에서 SVO 단문 3연속 이상 나열 없는가? SF01-SF05 위반 없는가? (`style-guide.md` 문장 단편화 방지 규칙 참조). 특히: (1) 체언 종결 파편("흔들었다. 작게.") → 수식어 통합, (2) 부정-긍정 이중 서술("X 아니었다. Y였다.") → 직접 서술, (3) 동시/연속 동작 분리("내렸다. 꺼냈다.") → ~면서/~한 뒤 접속, (4) 수치적 동작 나열("세 동작. 두 번 만에.") → 인물 특성 서술, (5) 설명 꼬리표("그것을 알았다.") → 삭제

## Daily Routine

### 시작 (10분)
1. 대시보드 확인
2. 현재작업상태 읽기
3. 관련 파일 열기
4. 목표 설정

### 집필 시간
1. 씬 체크리스트 확인
2. 목표 단어수 작성
3. 일관성 유지
4. 감정선 추적

### 마무리 (15분)
1. 진행상황 업데이트
2. 파일 백업 (Git commit)
3. 내일 계획

## Version Naming

```
ch-[NNN]_v[N]_[상태].md

상태: 초고 | 1차수정 | 2차수정 | 베타리딩 | 최종
예시: ch-001_v1_초고.md → ch-001_v2_1차수정.md
```
