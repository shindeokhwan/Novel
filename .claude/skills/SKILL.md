# Humanization Review - AI 인간화 검증 스킬

원고의 AI 흔적을 체계적으로 검출하고 인간화 점수를 산출하는 워크플로우.

## Trigger
- "인간화 리뷰", "AI 흔적 검사", "humanization check", "AI 지문 검사"

## Workflow

### Phase 1: Reference Loading (참고 자료 로드)
1. `08-피드백/AI흔적분석_리포트_2026-02-22.md` 로드
2. `.claude/rules/humanization-rules.md` 로드
3. 대상 범위 확인 (전체/특정 챕터/특정 씬)

### Phase 2: Pattern Scan (기계적 검출)
1. Grep: `0\.\d+초` → 수치적 시간 표현 검출
2. Grep: `입꼬리` → 반복 표현 검출
3. Grep: `동전.{0,10}(양면|닢)` → 동전 비유 반복
4. Grep: `퍼즐.{0,10}(조각|맞)` → 퍼즐 비유 반복
5. Grep: `파문.{0,10}(퍼|일)` → 파문 비유 반복
6. Grep: `모니터.{0,10}빛` → 3종 세트 (시각)
7. Grep: `팬.{0,5}소리|키보드.{0,5}소리` → 3종 세트 (청각)
8. Grep: `커서.{0,5}깜빡` → 3종 세트 (시각2)
9. Grep: `눈빛.{0,5}흔들` → 반복 표현
10. Grep: `어깨.{0,5}(축|처)` → 반복 표현
11. 각 에피소드 마지막 3줄 추출 → 종결 패턴 분류

### Phase 3: Depth Analysis (심층 분석)

#### 3A. 대화 자연스러움
- 정보전달형 대사 비율 산출
- 대화 노이즈 존재 여부 (끊김/딴소리/비완성)
- 캐릭터별 대사 추출 → 음성 구분도 평가

#### 3B. 갈등 해결 속도
- 주요 갈등 제기-해결 간격 측정
- 해결 후 잔여 긴장 존재 여부
- 갈등 깊이 규칙 준수 여부

#### 3C. 구조 다양성
- 에피소드 구조 패턴 분류 (시작-전개-종결 유형)
- 성장 곡선의 비균일성 (정체/후퇴 포인트 존재)
- 비합리적 행동/감정 존재 여부

#### 3D. 감각 다양성
- 씬당 사용된 감각 종류 카운트
- 동일 감각 조합 연속 사용 횟수
- 환경 묘사의 변주 정도

### Phase 4: Scoring (점수 산출)

| 항목 | 만점 | 채점 기준 |
|------|------|----------|
| 문체 다양성 | 20 | 반복 표현 <3회(+5), 비유 변주(+5), 문장 리듬(+5), 종결 다양(+5) |
| 대화 자연스러움 | 20 | 노이즈 >20%(+5), 정보전달 <10%(+5), 음성 구분(+5), 서브텍스트(+5) |
| 구조 비정형성 | 20 | 구조 5종+(+5), 종결 비연속(+5), 성장곡선 비균일(+5), 실패ep 존재(+5) |
| 감정 깊이 | 20 | 갈등 3ep+(+5), 비합리 행동(+5), 잔여 긴장(+5), 감정 모순(+5) |
| 감각 다양성 | 20 | 5종 감각(+5), 조합 비연속(+5), 신체반응 다양(+5), 환경변주(+5) |

### Phase 5: Report Generation (리포트 생성)

```
# 인간화 리뷰 리포트 - [날짜]
## 검사 범위: [전체/챕터 X-Y]

### 인간화 점수: X/100
- 문체 다양성: X/20
- 대화 자연스러움: X/20
- 구조 비정형성: X/20
- 감정 깊이: X/20
- 감각 다양성: X/20

### 검출된 AI 지문
[패턴별 발견 목록 + 위치 + 인용]

### 수정 제안 (Before/After)
[우선순위별 수정 제안]

### 이전 대비 변화
[이전 리뷰 대비 개선/악화 항목]
```

리포트를 `08-피드백/인간화리뷰_[날짜].md`에 저장.

## Output
- 인간화 점수 (0-100) + 영역별 점수
- 검출된 AI 지문 목록
- Before/After 수정 제안
- 저장된 리포트 경로
