# Entertainment Review - 재미평가 리뷰 스킬

3인 독자 페르소나(일반 독자, 장르 독자, 편집자)의 교차 분석으로 원고의 재미를 다각 진단한다.
핵심 진단 영역: 역사/바둑 접근성, 패턴 반복, 다이나믹/놀라움.

## Trigger
- "재미평가", "재미 리뷰", "entertainment review", "쾌감 밀도 체크", "재미 분석"

## Agents
- `general-reader` — 일반 독자 (25-35세, 역사/판타지 웹소설 가벼운 독자)
- `genre-reader` — 삼국지/회귀물 장르 독자 (30-40세, 삼국지·무협·회귀물 매니아)
- `editor-critic` — 편집자/비평가 (10년+ 경력)

## Workflow

### Phase 1: Context Loading (컨텍스트 로드)

1. 범위 결정: 사용자에게 확인
   - **전체**: 전 챕터 (시간 많이 소요)
   - **Part 단위**: Part 1(Ch.1-15) / Part 2(Ch.16-65) 등
   - **챕터 단위**: 특정 챕터 범위 (예: Ch.36-42)
2. 참조 파일 로드:
   - `.claude/reference/entertainment-density.md` — 쾌감 8패턴, 밀도 가이드
   - `.claude/reference/style-guide.md` — 톤 배합, 전투씬, 바둑 기풍
   - `.claude/reference/ability-upgrade-current.md` — AT1-AT4 업그레이드 빈도/트랙
   - `03-플롯/서사줄기추적.md` — 줄기 비율 데이터
   - `03-플롯/메인플롯-상세/PartN.md` — 챕터별 계획
   - `03-플롯/쾌감카운터.md` — 쾌감 패턴 누적 카운터
3. 범위 내 원고 파일 목록 확인 (Glob)

### Phase 2: 독립 분석 (Sub-agent ×3 병렬)

3개 에이전트를 병렬 실행. 각 에이전트에게 Phase 1에서 로드한 참조 파일 정보와 원고 범위를 전달.

#### 2A. general-reader (일반 독자)
- 에피소드별 몰입도 곡선 (1-5점)
- 스킵 충동 구간 마킹 (HISTORY/BADUK/REPEAT/SLOW/DISCONNECT)
- 감정 이입 피크/밸리
- 역사/바둑 접근성 평가 (1-5점)
- 유머 효과 평가 (HIT/SMILE/FLAT/CRINGE)
- TOP 5 재미 장면 / TOP 5 지루 장면
- 전진력 평가

#### 2B. genre-reader (장르 독자)
- 회귀물 설정별 신선도 (1-5점)
- 시대고증·전략 체계별 만족도 (1-5점)
- 전투·바둑 묘사 만족도 (체험적/설명적 분류)
- 전략-감정 결합도 (FUSION/LINKED/PARALLEL/SEPARATE)
- 삼국지 인물 생동감 평가
- 성장물 리듬 (1-5점)
- 전투씬 만족도 (1-5점)
- 바둑-전략 매핑 정합성 (1-5점)
- 장르 벤치마크 비교 (5작품)
- TOP 5 장르적 성취 / TOP 5 장르적 약점

#### 2C. editor-critic (편집자)
- 에피소드별 페이싱 곡선 (1-10점)
- 패턴 반복 감사:
  - 대화 패턴 5유형 (A-E) 분류 + 3연속 감지
  - 구조 패턴 5유형 (포석/돌파/여운/인간/전환) 분류 + 3연속 감지
  - 감정 리듬 3유형 (R/F/W) 분류 + 3연속 감지
- 서사 경제성 (씬별 기능 카운트)
- 캐릭터 활용도 매트릭스
- 쾌감 밀도 감사 (entertainment-density.md 대비)
- 능력 업그레이드 감사 (ability-upgrade-current.md 대비)
- TOP 5 강점 / TOP 5 약점

### Phase 3: 교차 토론 (종합 분석)

3개 에이전트 결과를 수신한 후 종합 분석.

#### 3A. 합의 영역
3인 모두 동의하는 강점과 약점 추출.
- 3인 모두 높은 평가를 준 장면 = **핵심 강점**
- 3인 모두 낮은 평가를 준 장면 = **핵심 약점**

#### 3B. 불일치 영역
관점 차이가 큰 지점을 식별하고 분석.
- 예: 역사 설명 — 일반 독자 "과잉(스킵)" vs 장르 독자 "적절(만족)"
- 예: 바둑 비유 — 일반 독자 "이해 불가" vs 장르 독자 "정밀하고 만족"
- 불일치마다 각 관점의 근거를 정리하고, 균형점 제안

#### 3C. 가중 투표
각 페르소나가 "가장 시급한 문제" 3개를 선택.
가중치: 일반 독자 ×1.5 / 편집자 ×1.2 / 장르 독자 ×1.0
(일반 독자 가중치 최고 = 대중성이 최우선)

투표 결과로 **핵심 약점 TOP 5** 확정.

### Phase 4: 해결안 제안

핵심 약점 TOP 5 각각에 대해:

1. **3인 각자의 해결안**
   - 일반 독자: "이렇게 바꾸면 더 읽고 싶어질 텐데"
   - 장르 독자: "삼국지/회귀물적으로 이 방향이 더 효과적"
   - 편집자: "구조적으로 이렇게 수정하면"

2. **종합 권고**: 3개 관점을 절충한 최적 해결안

3. **Before/After 예시**: 해당 약점의 구체적 원고 구간에 대한 수정 전/후 방향 제시

4. **수정 메타데이터**:
   - 난이도: 쉬움(문장 수정) / 보통(씬 수정) / 어려움(구조 재배치)
   - 영향 범위: 해당 씬만 / 챕터 전체 / 복수 챕터 / Part 전체
   - 우선순위: 즉시(P0) / 단기(P1) / 중기(P2)

### Phase 5: 종합 리포트 생성

#### 5차원 채점 (5×20 = 100점)

| 항목 | 만점 | 채점 기준 |
|------|------|----------|
| 몰입도 | 20 | ep 평균 몰입 ≥3.5(+5), 지루함 ep <10%(+5), 전진력 평균 ≥4(+5), Part당 하이라이트 ≥3(+5) |
| 접근성 | 20 | 역사/바둑 이해 평균 ≥3.5(+5), 스킵 충동 ep <8%(+5), 비유 성공률 ≥70%(+5), 맥락 전달 ≥80%(+5) |
| 다이나미즘 | 20 | 대화패턴 3연속 =0(+5), 구조패턴 3연속 =0(+5), 감정리듬 예측도 =낮음(+5), 놀라움 ≥3장면/Part(+5) |
| 삼국지 만족도 | 20 | 설정 신선도 평균 ≥3.5(+5), 전투·바둑 묘사만족 평균 ≥3.5(+5), 전략-감정 FUSION ≥3장면/Part(+5), 인물 생동감 ≥3/5(+5) |
| 구조적 견고함 | 20 | 처짐(페이싱 ≤3) ep =0(+5), 서사 경제성 ≥85%(+5), 쾌감밀도 위반 =0(+5), 능력 업그레이드 빈도 준수(+5) |

## Notes
- Phase 2 병렬 실행 시 각 에이전트에게 원고 전문과 참조 파일을 모두 전달할 것
- 원고 범위가 넓을 경우(20ep+) 에이전트당 컨텍스트 소진 가능 — Part 단위 분할 권장
- Phase 3-5는 메인 에이전트가 직접 수행 (교차 분석은 3개 결과를 동시에 볼 수 있어야 함)
- 이전 리포트가 있으면 로드하여 "이전 대비 변화" 섹션 추가
