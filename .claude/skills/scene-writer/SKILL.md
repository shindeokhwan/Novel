# Scene Writer - 씬 작성 스킬

## Trigger
- "씬 작성", "장면 쓰기", "씬 쓰기", "scene write"

## Workflow

### Phase 1: Context Loading
1. `00-핵심관리/현재작업상태.md` 읽기
2. 이전 2-3 에피소드 원고 읽기
3. 등장 캐릭터 파일 + 감정 상태 확인
4. `01-세계관/역사배경.md`에서 현재 시점 역사적 맥락
5. `03-플롯/서사줄기추적.md` → 이 씬의 줄기 결정
6. `03-플롯/복선추적.md` → 설치/회수할 복선
7. 바둑-전략 현재 단계 확인

### Phase 2: Pre-Write Validation
1. `01-세계관/세계관규칙.md` 관련 부분 확인
2. `01-세계관/타임라인.md`에서 현재 시점 확인
3. `03-플롯/씬체크리스트.md` 로드
4. 씬 목표: POV, 갈등, 목표, 변화
5. 무예/기 수준 확인

### Phase 3: Writing
1. 훅(Hook): 몰입 유도 시작
2. 설정(Context): 빠른 오리엔테이션
3. 상승(Build): 긴장감 고조
4. 절정(Climax): 핵심 순간/결정
5. 여파(Resolution): 즉각적 결과
6. 전환(Transition): 다음 씬 연결

### Phase 4: Post-Write Checklist
- [ ] POV 일관성
- [ ] 감각 묘사 3개+
- [ ] 시대고증 준수
- [ ] 대화 스타일 일관
- [ ] 바둑 용어 적절성
- [ ] 무예 수준 일관성
- [ ] 감정 연속성
- [ ] 마크다운 없음

### Phase 5: File Updates
1. `00-핵심관리/진행상황.md` 업데이트
2. 감정 추적 기록
3. 타임라인 업데이트
4. 복선 추적 업데이트
5. 서사줄기추적 비율 기록
