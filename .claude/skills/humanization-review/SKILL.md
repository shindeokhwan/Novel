# Humanization Review - AI 인간화 검증 스킬

원고의 AI 흔적을 체계적으로 검출하고 인간화 점수를 산출하는 워크플로우.

## Trigger
- "인간화 리뷰", "AI 흔적 검사", "humanization check", "AI 지문 검사"

## Workflow

### Phase 1: Reference Loading (참고 자료 로드)
1. `08-피드백/` 내 최신 AI 흔적 분석 리포트 로드
2. `.claude/reference/humanization-rules.md` 로드
3. `.claude/reference/style-guide.md` 로드 (SF01-SF05, em-dash 규칙)
4. 대상 범위 확인 (전체/특정 챕터/특정 씬)

### Phase 2: Pattern Scan (기계적 검출)

#### 2A. 시대착오어 검출 (humanization-rules.md 절대 금지 11종)
1. Grep: `타이밍` → 시대착오어 (대체: 시기/때/순간)
2. Grep: `리듬` → 시대착오어 (대체: 박자/흐름)
3. Grep: `패턴` → 시대착오어 (대체: 수순/습관/결)
4. Grep: `시스템` → 시대착오어 (대체: 체계/제도)
5. Grep: `스타일` → 시대착오어 (대체: 방식/방법)
6. Grep: `카리스마` → 시대착오어 (대체: 위엄/덕망)
7. Grep: `퍼센트|%` → 시대착오어 (대체: 완전히/온전히)
8. Grep: `이미지` → 시대착오어 (대체: 모습/형상)
9. Grep: `레벨` → 시대착오어 (대체: 수준/단계)
10. Grep: `에너지` → 시대착오어 (대체: 기력/힘)
11. Grep: `포인트` → 시대착오어 (대체: 요점/핵심)

#### 2B. AI 지문 패턴 검출
12. Grep: `0\.\d+초` → 수치적 시간 표현 검출
13. Grep: `입꼬리` → 반복 표현 검출
14. Grep: `눈빛.{0,5}흔들` → 반복 표현
15. Grep: `어깨.{0,5}(축|처)` → 반복 표현
16. Grep: `말했다` → 누적 카운트 (챕터당 3회 초과 시 경고)
17. Grep: `—` → em-dash 카운트 (챕터당 5회 초과 시 경고)

#### 2C. SVO 단문 연쇄 검출
18. 각 에피소드를 문장 단위로 분리 → 20자 이하 문장이 3개 이상 연속되는 구간 검출
    - 의도적 스타카토 여부 판정 (긴장/감각/결단 맥락인지 확인)
    - 비의도적 단문 연쇄 = P2 수정 대상으로 분류

#### 2D. 분리호칭 대화 패턴 검출
19. Grep: `"[^"]{1,5}"\s*[가-힣]{1,4}(이|가)\s*말했다` → 분리호칭 패턴 의심 구간 추출
    - 이름/호칭만 단독 인용 후 "말했다" + 별도 인용부호로 내용 이어지는 패턴

#### 2E. 종결 패턴 분류
20. 각 에피소드 마지막 3줄 추출 → 종결 패턴 분류 (끊김형/대화잔향형/감각단독형/질문형/타인시선형)
    - 직전 에피소드와 동일 종결 = P1 수정 대상

### Phase 3: Depth Analysis (심층 분석)

#### 3A. 대화 자연스러움
- 정보전달형 대사 비율 산출
- 대화 노이즈 존재 여부 (끊김/딴소리/비완성)
- 캐릭터별 대사 추출 → 음성 구분도 평가
- 시대적 대화 다양성: 모사형(우회적)/무장형(직설적)/군주형(위엄)/내면형(현대적) 적절한 분포
- `"[대사]." [이름]이/가 말했다.` 단독 패턴 검출 (수식어/행동 비트 없는 형태)

#### 3B. 갈등 해결 속도
- 주요 갈등 제기-해결 간격 측정
- 해결 후 잔여 긴장 존재 여부
- 갈등 깊이 규칙 준수 여부

#### 3C. 구조 다양성
- 에피소드 구조 패턴 분류 (포석/돌파/여운/인간/전환)
- 성장 곡선의 비균일성 (정체/후퇴 포인트 존재)
- 비합리적 행동/감정 존재 여부

#### 3D. 감각 다양성
- 씬당 사용된 감각 종류 카운트
- 동일 감각 조합 연속 사용 횟수
- 삼국시대 특화 감각 (전고, 말발굽, 검, 바둑돌 등) 활용도
- 환경 묘사의 변주 정도

### Phase 4: Scoring (점수 산출)

| 항목 | 만점 | 채점 기준 |
|------|------|----------|
| 문체 다양성 | 20 | 반복 표현 <3회(+5), 비유 변주(+5), 문장 리듬: 리듬 파형 다양성 + 단문 밀도 위반 0건(+5) / 1-2건(+3) / 3건+(+0), 종결 다양(+5) |
| 대화 자연스러움 | 20 | 노이즈 >20%(+5), 정보전달 <10%(+5), 음성 구분(+5), 서브텍스트(+5) |
| 구조 비정형성 | 20 | 구조 5종+(+5), 종결 비연속(+5), 성장곡선 비균일(+5), 실패ep 존재(+5) |
| 감정 깊이 | 20 | 갈등 3ep+(+5), 비합리 행동(+5), 잔여 긴장(+5), 감정 모순(+5) |
| 감각 다양성 | 20 | 5종 감각(+5), 조합 비연속(+5), 신체반응 다양(+5), 환경변주(+5) |

**시대착오어 감점**: 시대착오어 1건당 총점에서 -2점 (최대 -20점)

### Phase 5: Report Generation (리포트 생성)

```
# 인간화 리뷰 리포트 - [날짜]
## 검사 범위: [전체/챕터 X-Y]

### 인간화 점수: X/100
- 문체 다양성: X/20
- 대화 자연스러움: X/20
- 구조 비정형성: X/20
- 감정 깊이: X/20
- 감각 다양성: X/20
- 시대착오어 감점: -X점

### 검출된 AI 지문
#### 시대착오어
[단어 | 위치 | 원문 | 대체 제안]

#### SVO 단문 연쇄
[위치 | 원문 | 의도적/비의도적]

#### 분리호칭 패턴
[위치 | 원문 | 수정 제안]

#### em-dash 남용
[챕터 | 횟수 | 위치 목록]

#### 기타 AI 지문
[패턴별 발견 목록 + 위치 + 인용]

### 수정 제안 (Before/After)
[우선순위별 수정 제안]

### 이전 대비 변화
[이전 리뷰 대비 개선/악화 항목]
```

리포트를 `08-피드백/인간화리뷰_[날짜].md`에 저장.

## Output
- 인간화 점수 (0-100) + 영역별 점수
- 검출된 AI 지문 목록 (시대착오어, SVO, 분리호칭, em-dash 별도 분류)
- Before/After 수정 제안
- 저장된 리포트 경로
