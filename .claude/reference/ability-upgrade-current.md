# Ability Upgrade Rules - 현재 상태 + 레퍼토리 + 빈도 규칙

주인공 서홍(서진우)의 4개 능력 트랙 관리, 소소한 레벨업 에피소드 설계 원칙.
매 3-4챕터에 1회 이상 업그레이드 순간을 배치하여 독자의 성장 체감을 보장한다.

**전체 이력**: `.claude/reference/ability-upgrade-history.md` 참조

## 4개 능력 트랙 (Ability Tracks)

| ID | 트랙 | 설명 | 주요 표현 방식 |
|----|------|------|--------------|
| AT1 | 무력(武力) | 기(氣) 단계, 검술 숙련도, 전투 직감, 체력/반응속도 | 전투 장면에서 "이전에 못 했던 것이 됨" |
| AT2 | 지력(智力) | 바둑 수읽기 깊이, 대국관, 전략 차원, 인물 심리 독해 | 내면 독백에서 "인식의 층위가 올라감" |
| AT3 | 통치력(統治力) | 행정 효율, 백성 신뢰, 법제/경제, 세력 운영 | 결과로 보여짐 — "전과 달리 민심이 ~" |
| AT4 | 충성심(忠誠心) | 수하들의 개인적 신뢰, 집단 결속력, 자발적 희생 의지 | 타인의 행동으로 보여짐 — 수하가 먼저 움직임 |

### 기(氣) 단계 - AT1 기준선 (변경 불가)

| 단계 | 달성 챕터 | 특이사항 |
|------|----------|---------|
| 감기(感氣) | Ch.9 확정 | 소 업그레이드는 감기 숙련 과정에서 다수 허용 |
| 운기(運氣) | Ch.22 확정 | 달성 전 소 업그레이드 2-3회 포석 가능 |
| 발기(發氣) | Ch.58 확정 | Ch.45 부상 후 화타 18개월 치료→Ch.58 돌파. 실리형 발기(좁은 기맥+밀도 높은 순환) |
| 통기/화기 | 미달성 | 주인공 무예 제한 — 발기가 한계, 전략으로 보완 |

## 빈도 규칙 (Frequency Rules)

```
3-4챕터당: 소 업그레이드 최소 1회 (4개 트랙 합산)
10챕터당:  중 업그레이드 최소 1회
소아크 종결: 대 업그레이드 최소 1회 (중 이상)
```

### 경고 임계값
- **5챕터 연속 업그레이드 0회**: 독자 성장 체감 소실 경고 → 즉시 소 업그레이드 삽입
- **같은 트랙만 3회 연속**: 다른 트랙으로 전환 필요 (트랙 균형)
- **무력만 집중**: 웹소설 무투물화 경고 → 지력/통치력 업그레이드 보강

## 작성 원칙 (Writing Principles)

### 필수 4요소 (최소 2개)
1. **Before/After 대비**: 독자가 변화를 느낄 수 있어야 한다
2. **주인공 자기 인식**: 내면 독백 또는 행동으로 표현
3. **타인 반응**: 수하, 적, 제3자의 새로운 반응
4. **구체적 결과**: 씬/챕터 내 구체적 결과

### 금지 패턴
- "갑자기 강해졌다/똑똑해졌다" 서술 금지
- 수치 직접 기재 금지: "힘이 10% 올랐다"
- 대 업그레이드 두 개를 한 챕터에 배치 금지
- 기(氣) 단계 무단 상향 금지
