# World Building Rules - 세계관 구축 규칙

세계관 일관성 관리, 절대/유연 규칙, 역사 검증 기준, 무예 체계를 정의한다.

## Absolute Rules (변경 불가)

### 역사적 대세
- 주요 역사 사건(황건적의 난, 동탁의 난, 관도대전, 적벽대전 등)은 발생함
- 단, 주인공의 개입으로 결과와 전개가 달라질 수 있음
- 실존 인물의 기본 성격 골격 유지 (조조의 야심, 유비의 인덕 등)
- 이미 죽은 역사적 인물을 살리는 것은 복선 필요

### 기(氣) 체계
- 5단계: 감기(感氣) → 운기(運氣) → 발기(發氣) → 통기(通氣) → 화기(化氣)
- 단계 상승에는 수련 시간 필요 (단기 비약 금지)
- 기 수련은 선천적 재능 + 후천적 노력
- 주인공은 무예 후발주자 (운기~발기 한계, 전략으로 보완)

### 회귀 규칙
- 단일 회귀 (재회귀 불가, 세이브/로드 없음)
- 현대 기억 완전 보존
- 역사 지식은 삼국지연의 수준 (정사와 차이 있음)
- 현대 기술 직접 재현 불가 (재료/인프라 부재)
- 개념적 응용만 가능 (위생, 병법 아이디어 등)
- 회귀 원인은 미스터리로 남김 (AI 메커니즘 미공개, 순수 삼국지 통일에 집중)

### 바둑-전략 법칙
- 주인공의 전략은 바둑적 사고에 기반해야 함
- 바둑 용어 사용 시 실제 바둑 개념과 정확히 대응

### 물리 법칙
- 기본 물리 법칙 준수 (기(氣)에 의한 초인적 능력은 허용)
- 무기/갑옷의 물리적 한계 존중
- 이동 시간/거리의 현실적 계산

## Flexible Rules (조정 가능)

### 무명 인물
- 역사에 기록되지 않은 인물 자유 창작
- 지방 관리, 호족, 상인, 의사 등

### 지방 사정
- 각 지역의 세부 정치 상황 조정 가능
- 호족/토착 세력 자유 설정
- 지방 관리의 성향 창작

### 전투 디테일
- 극적 윤색 허용
- 실제 전법 기반 + 예술적 과장
- 일기토(一騎討) 허용 (과도한 무협화 금지)

### 기(氣) 세부 발현
- 단계별 세부 능력 조정 가능
- 특수 무기/기법과의 결합 창작

## 무예/내공 등급 체계

| 등급 | 기(氣) 단계 | 대표 인물 (삼국지) | 전투력 |
|------|-----------|-------------------|--------|
| 초절정 | 화기(化氣) | 여포 | 1인으로 천 명 상대 |
| 절정 | 통기(通氣) | 관우, 장비, 조운 | 1인으로 백 명 상대 |
| 일류 상위 | 발기(發氣) 숙련 | 장합, 장료 | 절정 근접, 1인으로 수십 명+ 상대 |
| 일류 | 발기(發氣) | 감녕, 황충 | 1인으로 수십 명 상대 |
| 이류 | 운기(運氣) | 일반 장수급 | 1인으로 수 명 상대 |
| 삼류 | 감기(感氣) | 뛰어난 병사 | 일반 병사 수 명 |
| 일반 | 무기(無氣) | 보통 병사 | 개인 전투력 |

### 주인공 무예 제한
- 시작: 무기(無氣) — 전신 마비 후유증, 몸 적응 필요
- Part 1 끝: 감기(感氣) — 기 감지 시작
- Part 2 중반(Ch.58): 발기(發氣) — Ch.45 기맥 손상 후 화타 치료로 회복·돌파
- Part 2 끝~Part 3: 발기 유지 — 한계 (이류~일류 경계)
- Part 4: 발기 유지 — 무예가 아닌 전략으로 천하 통일

### 사상자 보고 규칙
- **현령급**: 전사자 이름 개별 추적 가능 (수십 명 단위)
- **태수급**: 지휘관급(오장 이상) 이름 추적 + 총 수치 보고
- **주목급 이상**: 총 수치 보고 중심. 감정적 무게는 '총 수의 무게'로 표현. 지휘관급 전사자만 개별 언급
- 단, 주목이 되어도 "명부를 읽는" 장면은 지휘관급 명단에 한정하여 허용

### 별동대/파견군 수치 일관성
- 별동대 총수 = 선봉 + 본대 (합산이 정확히 맞아야 함)
- 전투 후: 사살 + 항복 + 도주 = 교전 병력 (대략 맞아야 함)

### 연의 병력 vs 실제 병력 (필수 고시)
- 삼국지연의의 병력 수치는 대폭 과장되어 있음. 서홍은 연의를 기억하므로 이 괴리를 인식해야 함
- 연의 "칠십만" → 실제 십만 수준. 연의 "백만" → 실제 이십만 이하
- **작중 병력은 역사적 실제에 가까운 수치를 사용** (연의 수치가 아님)
- 서홍이 관도대전 등 대규모 전쟁을 처음 접할 때, "연의에서는 수십만이라 했으나 실제로는 수만" 이라는 내면 독백으로 독자에게 고시할 것
- 이 고시는 Part 3 초반(관도대전 진입 시점)에 1회 명시적으로 넣고, 이후에는 자연스럽게 실제 수치만 사용

## SF Science Validation → 역사/무예 Validation Checklist

### 역사 검증
- [ ] 사건의 시기가 타임라인과 일치
- [ ] 실존 인물의 위치가 역사적으로 가능
- [ ] 관직/작위가 시기에 맞음
- [ ] 기술 수준이 시대에 부합 (의식주, 무기, 건축)

### 무예 검증
- [ ] 캐릭터의 무예 수준이 기(氣) 단계와 일치
- [ ] 전투 결과가 무예 수준 차이에 부합
- [ ] 주인공의 무예 성장이 시간/수련량과 비례
- [ ] 전략/지략이 무예 부족을 보완하는 구도 유지

### 지리 검증
- [ ] 도시/지역 간 이동 시간의 합리성
- [ ] 지형이 전투/전략에 미치는 영향 반영
- [ ] 계절/날씨의 일관성
