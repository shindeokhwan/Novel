# Core Rules - Sub-Agent 집필용 핵심 규칙

Phase 3 병렬 집필 Sub-agent에게 자동 주입되는 규칙 요약본.
전체 규칙은 `.claude/reference/` 하위 파일 참조. 이 파일은 집필 시 필수 최소 규칙만 포함한다.

---

## 1. 산문 형식

- 마크다운 형식 절대 금지 (순수 산문만)
- POV: 3인칭 제한 시점, 씬 내 POV 전환 절대 금지
- 시제: 과거형
- 톤: 웅장한 서사(40%) + 전략적 긴장(35%) + 인간적 따뜻함(25%)
- 서술형 완결문 기본. 파편적 단문 나열 금지 (연속 2문장 이상 단문 금지)
- 내면 묘사 풍부하게 — 판단 근거, 감정 흐름을 서사로 채운다
- 바둑/전략 묘사 상세하게 — 독자가 바둑을 몰라도 이해하도록
- 씬 전환: `* * *`
- 대화: 큰따옴표

## 2. 절대 금지 패턴

### 시대착오 단어 (11종 — 대사·서술·내면 막론 금지)
| 금지 | 대체 |
|------|------|
| 타이밍 | 시기/때/순간 |
| 리듬 | 박자/흐름 |
| 패턴 | 수순/습관/결 |
| 시스템 | 체계/제도 |
| 스타일 | 방식/방법 |
| 카리스마 | 위엄/덕망/풍모 |
| 퍼센트(%) | 완전히/온전히 |
| 이미지 | 모습/형상 |
| 레벨 | 수준/단계 |
| 에너지 | 기력/힘 |
| 포인트 | 요점/핵심 |

### AI 흔적 최상위 패턴
- **SVO 단문 연쇄**: "SVO. SVO. VO." 3문장 이상 연속 절대 금지
  - SF01: 체언 종결 파편("흔들었다. 작게.") → 수식어 통합
  - SF02: 부정-긍정 이중 서술("X 아니었다. Y였다.") → 직접 서술
  - SF03: 동시/연속 동작 분리("내렸다. 꺼냈다.") → ~면서/~한 뒤 접속. 단, 대상 문장에 이미 전환(-었다가)이 있으면 통합 금지
  - SF04: 수치적 동작 나열("세 동작. 두 번 만에.") → 인물 특성으로 서술
  - SF05: 설명 꼬리표("그것을 알았다.") → 이미 전달된 정보면 삭제
- **분리 호칭 패턴**: `"조우." 서홍이 말했다. "내용"` → 합치거나 행동 비트로 전환
- **`"[대사]." [이름]이/가 말했다.` 단독 패턴**: 수식어 없이 대사+말했다만 → 태그 삭제, 행동 비트, 또는 다른 동사(답했다/받았다/덧붙였다)
- **수치 시간**: "0.1초", "0.3초간" → 체감 표현("심장이 한 번 뛰는 시간")
- **챕터 번호 참조**: "Ch.X에서" → 구체적 시간/장소/사건으로 대체
- **BCI 현재형 금지**: "BCI 뇌가 분석하는" → BCI는 과거 기억으로만 참조
- **메타 서술 라벨**: "수읽기가 시작됐다", "플래시백" → 라벨 없이 직접 침잠

### em-dash(—) 규칙
- 챕터당 최대 5회, 같은 문단 2회 이상 금지
- 접속사 직후 em-dash 금지, 열거 총괄 금지, 삽입어구 쌍 금지

## 3. 대화 규칙

### 시대적 대화 스타일
- 고어체 기반: "~하시오", "~이로다", "~하겠소" (현대적 가독성 유지)
- 주인공 내면: 현대어 (바둑 비유, 현대 비유)
- 무장: 호탕, 직설 / 모사: 우회, 비유 / 군주: 위엄+인간미

### 필수 대화 노이즈 (챕터당 최소 3회)
끊김 / 딴소리 / 비완성 문장 / 오해 / 침묵 중 3종 이상

### 대화 비율
대화 ~35% / 행동·묘사 ~40% / 내면 묘사 ~25%

### 금지
- 정보전달형 대사 (상대가 아는 정보를 독자를 위해 설명)
- "말했다" 수식어 없이 단독 사용 (위 참조)

## 4. 감각 묘사

- 매 씬 최소 3개 감각, 시각 편중 금지
- 삼국시대 감각: 깃발/갑옷/화염/전고/말발굽/검 자루/피/솔밭/거친 밥
- 감각 재발견 (주인공): 전신 마비 → 건강체의 감격. 위기 시 재부각

## 5. Show Don't Tell

- 감정 직접 서술 금지 → 신체 반응으로
- 전략 직접 서술 금지 → 바둑판 비유로
- 위기씬: 분석적 거리 금지 → 즉각적 패닉(파편적 사고, 딜레마)

## 6. 분량

- 에피소드(씬) 1개: 3,000~4,000자 목표, 최소 2,000자
- 챕터 전체(3씬): 9,000~12,000자, 최소 7,000자

## 7. 에피소드 종결 방식 (5가지 순환)

1. 끊김형 / 2. 대화 잔향형 / 3. 감각 단독형 / 4. 질문형 / 5. 타인 시선형
→ 아웃라인에 지정된 종결 유형을 따른다. 직전 에피소드와 동일 방식 금지.

## 8. 비합리성 쿼터

챕터당 최소 1회: 비합리적 선택, 감정에 의한 실수, 자기기만, 회피, 과잉반응 중 하나.
