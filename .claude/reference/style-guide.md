# Style Guide Rules - 문체/스타일 규칙

삼국지 회귀물 "Thriple Kingdom"의 문체 기준, POV 규칙, 감각 묘사 기준을 정의한다.

## Narrative Style

### 기본 설정
- **POV**: 3인칭 제한 시점 (Multi-POV, 주인공 중심)
- **시제**: 과거형
- **톤**: 웅장한 서사(40%) + 전략적 긴장(35%) + 인간적 따뜻함(25%)
- **문체 수준**: 문학적 대중소설 — 나관중의 서사적 무게 + 웹소설의 흡인력 + 히카루의 바둑 묘사

### 산문 규칙
- 원고 내 마크다운 형식 절대 금지
- 자연스러운 단락 흐름으로 서술
- 능동태 우선
- **서술형 완결문 기본**: 모든 문장은 주어-서술어가 갖춰진 완결된 형태를 기본으로 한다
- **파편적 단문 나열 금지**: "흙이었다. 처음 느낀 것은 흙이었다." 같은 체언 종결, 단어 나열, 미완결 문장의 반복을 지양한다
- **예외**: 극적 강조가 필요한 순간(대사 직후, 절정의 한 문장)에만 단문 허용. 연속 2문장 이상 단문 나열 금지
- **내면 묘사 풍부하게**: 인물의 사고 과정, 판단 이유, 감정의 흐름을 충분히 풀어서 서술한다. 행간에 맡기지 말고 서사로 채운다
- **바둑/전략 묘사 상세하게**: 포석, 형세, 수읽기 등 전략적 판단의 근거와 고뇌를 구체적으로 서술한다. 독자가 바둑을 몰라도 이해할 수 있도록 풀어 쓴다

### 문장 단편화 방지 규칙 (Sentence Fragmentation Prevention)
*(Ch.040 리터칭 분석에서 도출 — 2026-02-26)*

행동 묘사에서 SVO 단문이 연속 나열되는 패턴은 AI 흔적이자 산문의 밀도를 떨어뜨리는 주요 원인이다. 아래 5개 하위 규칙을 적용한다.

#### SF01: 체언 종결 파편 → 수식어 통합
- **금지**: "고개를 흔들었다. 작게." — 수식어를 별도 파편 문장으로 분리
- **권장**: "고개를 작게 좌우로 흔들었다." — 수식어를 본문에 통합하고, 구체적 방향/양태 추가

#### SF02: 부정-긍정 이중 서술 삭제
- **금지**: "말에서 내리지 말라는 것이 아니었다. 소리 내지 말라는 것이었다." — "X가 아니다"로 한 문장 쓴 뒤 "Y이다"로 또 한 문장
- **권장**: "소리 내지 말라는 것이었다." — 의미를 직접 서술. 부정문은 독자에게 불필요한 오독 경로를 열었다 닫는 낭비
- **예외**: 인물이 상황을 오판했다가 정정하는 내면 독백(사고의 전환을 보여주는 경우)에만 허용

#### SF03: 동시/연속 동작 분리 금지 → 접속 통합
- **금지**: "말에서 내렸다. 창을 꺼냈다." / "끄덕였다. 고삐를 당겼다." — 동시 또는 즉각 연속인 동작을 각각 별도 문장으로 분리
- **권장**: "창을 꺼내면서 말에서 내렸다." / "이해했음을 표시한 뒤 조용히 고삐를 당겼다." — "~면서/~며/~한 뒤/~자마자" 등 접속어미로 동작 관계를 명시하고, 부사(조용히, 빠르게)로 양태 보충
- **판단 기준**: 두 동작 사이에 시간 간격이나 인식 전환이 없으면 통합 대상. 간격/전환이 있으면 분리 유지
- **통합 금지 조건 (안전장치)**: 아래 경우 SF03을 적용하지 않고 분리를 유지한다
  1. **대상 문장에 이미 방향 전환이 있을 때**: `-었다가`, `-려다가`, `-았으나` 등 문장 내부에 이미 동작 전환/중단이 포함된 경우, 앞 문장까지 통합하면 접속이 중첩되어 비문이 된다. 예: "생각했다. 입을 열었다가 방향을 바꾸었다." → 통합 금지 (두 번째 문장에 `-었다가` 전환 이미 존재)
  2. **통합 결과 접속어미 2개 이상 중첩**: 한 문장 안에 `-며`, `-면서`, `-다가`, `-자` 등 접속어미가 2개 이상 겹치면 가독성이 급락한다. 통합 전에 결과 문장의 접속어미 수를 확인한다
  3. **통합 후 음독 검증**: 통합한 문장을 소리 내어 읽었을 때 자연스럽지 않으면 원래 분리를 유지한다. 기계적 통합보다 자연스러운 한국어가 우선이다

#### SF04: 수치적 동작 나열 → 인물 특성 서술
- **금지**: "세 동작이 필요했다. 세 동작을 두 번 만에 했다." — 동작을 숫자로 분해하여 기계적으로 나열
- **권장**: "시간이 걸리는 일이었지만 숙련된 무인은 그 시간을 단축시켰다." — 수치 대신 인물의 특성(숙련됨, 경험, 습관)으로 행동의 의미를 전달
- **원칙**: 독자에게 중요한 것은 "몇 동작"인지가 아니라 "이 인물이 어떤 사람인지"이다

#### SF05: 설명 꼬리표 삭제
- **금지**: "두기는 그것을 알았다." / "소리가 없었다." — 앞 문장에서 이미 전달된 정보를 별도 문장으로 재확인
- **권장**: 이미 맥락으로 전달된 정보면 삭제. 필요하면 앞 문장의 부사구로 흡수 ("소리없이 도를 꺼냈다")
- **판단 기준**: 해당 문장을 삭제해도 독자가 같은 정보를 얻을 수 있으면 삭제 대상

#### 문장 단편화 자가 진단
한 문단 내에서 아래 패턴이 보이면 단편화를 의심하고 통합을 검토한다:
- **주어 생략 단문 3연속**: "SVO. VO. VO." — 같은 주어의 행동이 3문장 이상 분리
- **문장 평균 길이 10자 이하** 구간이 4문장 이상 지속
- **접속어미 0회** 구간이 5문장 이상 지속 (모든 문장이 마침표로만 종결)

### 원고 형식
```
순수 산문 형식:
- 단락은 빈 줄로 구분
- 대화는 큰따옴표 사용
- 내면 묘사는 이탤릭 또는 자연스러운 서술
- 씬 전환은 "* * *" 또는 빈 줄 3개
```

## 구두점 규칙 (줄표/Em-Dash)

### 사용 상한
- **챕터당 em-dash(—) 최대 5회**
- 같은 문단 내 em-dash 2회 이상 절대 금지
- 직전 1000자 이내 em-dash 재사용 금지

### 절대 금지 용법
- 접속사(그때, 그리고, 그런데, 하지만) 직후 em-dash
- 열거 총괄 용법: "A, B, C — 모두~" → 마침표+새문장 또는 쌍점 사용
- 삽입어구 쌍: "A — B — C" → 관형절, 쉼표, 문장 분리로 대체

### 허용 용법 (4조건 모두 충족 시에만)
1. 해당 문단 내 유일한 em-dash
2. 쉼표/마침표로 대체 시 의미 또는 리듬 훼손
3. 챕터의 핵심 반전/깨달음 순간
4. 직전 1000자 이내 다른 em-dash 없음

### 허용 예시
- **외부 요인 대화 중단**: 타인이 끼어들어 대사가 끊길 때
- **자기 교정**: 말하다 스스로 고치는 순간 (챕터당 1회 이내)
- **핵심 반전**: 챕터 클라이맥스의 결정적 깨달음 (1회)

### 대체 수단 분산 (패턴 남용 방지)
| 수단 | 비율 | 적합 상황 |
|------|------|----------|
| 마침표+새문장 | 35% | 삽입어구, 반전, 열거 |
| 문장 구조 재구성 | 25% | 삽입어구, 반전 |
| 쉼표/동격 | 20% | 삽입어구, 대화 중단 |
| 단락 분리(빈줄) | 10% | 반전/여운 |
| 말줄임표("...") | 5% | 대화 자발적 멈춤 (챕터당 2회 이내) |
| 단순 삭제 | 5% | 접속사 뒤 em-dash |

## POV Rules (3인칭 제한 시점)

### 필수 준수
- 씬 내 POV 전환 절대 금지
- POV 캐릭터의 감각/생각만 직접 서술 가능
- 다른 캐릭터의 내면은 추측/관찰로만 표현
- POV 캐릭터가 모르는 정보 서술 금지

### Multi-POV 규칙
- 주인공(서진우/장삼) POV: 전체의 70% 이상
- 적 진영/조연 POV: 30% 이하, 챕터 단위 전환만
- POV 전환 시 새 POV 캐릭터의 감각으로 재설정

## Sensory Description

### 최소 기준
- 매 씬 최소 3개 감각 활용
- 시각 묘사에만 편중 금지

### 삼국시대 감각 묘사
- **시각**: 깃발, 갑옷, 화염, 먼지, 별, 달
- **청각**: 전고(戰鼓), 말발굽, 검 부딪힘, 함성, 바람
- **촉각**: 검 자루, 말 등, 흙, 비단, 피, 바둑돌의 차가움
- **후각**: 피, 화약, 말, 솔밭, 술, 밥 짓는 냄새
- **미각**: 전장의 먼지, 술, 거친 밥, 약초

### 감각 재발견 (주인공 전용)
전신 마비였던 주인공이 건강한 몸을 얻은 감격. 초반부(Part 1)에서 집중 묘사:
- 풀을 밟는 발바닥의 감촉
- 검을 쥐는 손의 무게감
- 바람이 얼굴을 스치는 느낌
- 음식의 맛과 온기
- 달리는 근육의 움직임
- 점차 당연해지면서 줄어들되, 위기 순간 다시 부각

## Show Don't Tell

### 감정 표현
- **금지**: "그는 분노했다"
- **권장**: "주먹이 바둑돌처럼 단단하게 쥐어졌고, 턱 근육이 돌처럼 굳었다"

### 전략적 사고 표현
- **금지**: "그는 전략을 세웠다"
- **권장**: "머릿속에 바둑판이 펼쳐졌다. 적의 진형은 화점 포석, 좌하귀에 빈 곳이 있었다"

### 메타 서술 라벨 절대 금지
독자에게 현재 일어나는 일을 설명하는 라벨은 쓰지 않는다.
- **금지**: "그리고... 플래시백.", "수읽기가 시작됐다", "기억이 밀려왔다" (선언형)
- **권장**: 라벨 없이 직접 기억/감각/사고 속으로 침잠. 독자가 함께 빠져들게 한다.
- **원칙**: 무슨 일이 일어나고 있는지를 '안내'하지 않는다. 그냥 일어나게 한다.

### 챕터 번호 산문 참조 절대 금지
원고 내에서 챕터/화수를 번호로 참조하는 것은 작가의 존재를 드러내는 메타 위반이다.
- **절대 금지**: "Ch.X에서", "X화 때", "X챕터에서", "이전 회차에서" 등 번호 직접 기재
- **권장**: 구체적인 시간("사흘 전", "처음 합류한 날"), 장소("훈련장에서"), 사건("첫 전투에서")으로 대체
- **예시**: "Ch.7의 훈련장에서" → "훈련장에서", "Ch.1에서 부상자를" → "이틀 전 부상자를"
- **원칙**: 독자는 챕터 번호를 모른다. 사건 그 자체로 기억하게 한다.

## 위기·충격씬 내면 묘사 규칙
*(Ch.014 플래시백 씬 분석에서 도출 — 2026-02-23)*

### 원칙: 분석적 거리 → 즉각적 패닉

위기/충격 상황(전투, PTSD, 감기 과부하, 첫 살인)에서 주인공의 내면은 **냉정한 분석가**가 아니라 **패닉 상태의 생존자**로 묘사한다.

| 금지 (분석적 거리) | 권장 (즉각적 패닉) |
|---|---|
| "과부하. 머릿속이 하얘지는 순간이 왔다." | 감각 자체로 표현. 하얘지는 것을 서술하지 말고 독자가 하얗게 만들어라. |
| "그는 공포를 느꼈다." | 공포가 신체를 통해 드러나게. |
| 완결된 3문장 내면 독백 | 파편적 자기 수정 또는 단절된 질문 |

### 내면 독백: 수동적 공포 → 능동적 딜레마

- **금지 (수동)**: "이 감각이 사라지면." → 위협이 외부에서 오는 피해자 구조.
- **권장 (능동)**: 스스로 선택해야 하는 순간이 공포의 핵심. 예: 과부하를 끊으면 되지 않나 → 그게 곧 15년의 천장이다 → 딜레마.
- 위기 중 내면 독백은 질문으로 파편화한다. 완결된 분석문이 아니라 끊어지는 사고.
  - **금지**: "감각을 차단해야 하나? 감각을 차단한다고? 그럼 어떻게 되는거지?" (너무 정제됨)
  - **권장**: "감각을 — 차단한다고?" 또는 "차단하면. 그럼—" 식으로 압축·단절.

### BCI 표현 금지 (세계관 정합성)

삼국시대의 서진우 몸에서 BCI는 물리적으로 작동하지 않는다. BCI는 기억이자 정체성이지, 고대에서 구동되는 하드웨어가 아니다.

- **절대 금지**: "서진우의 BCI 뇌가 자동으로 분석하는 정보"
- **허용**: "BCI로 바둑을 두던 감각이 생각났다" (과거 기억으로 참조)
- **대체**: 분석적 사고 패턴은 "이 버릇은 어디서 온 것인지" 식의 자기 인식으로 처리

### 이미지 연쇄 유지 원칙

씬 내부에서 감각 이미지는 자연스럽게 이어져야 한다. 과부하 묘사의 색/감각이 이후 플래시백 이미지와 연결되어야 독자가 두 층위를 동시에 경험한다.

- **예시 (好)**: 과부하 → "머릿속이 하얘졌다" → 플래시백 → "하얀 천장". '하얀'이 두 공간을 연결.
- **주의**: 과부하 묘사 변경 시 이후 이미지와의 연쇄가 끊기지 않는지 확인.

### 설계된 수미상관 구조 제한

고긴장 씬에서 지나치게 설계된 수미상관은 '작가가 보인다'는 느낌을 준다.
- **고긴장 씬**: 수미상관 금지. 기억/감각이 무질서하게 지나가는 편이 자연스럽다.
- **성찰/독백 씬**: 수미상관 허용. 차분한 사고 흐름에서는 구조가 어울린다.

### 시간 표현 (위기씬 특화)

수치 시간은 humanization-rules.md의 절대 금지 사항이나, 위기씬에서 특히 주의한다.
- **절대 금지**: "0.5초간", "0.2~3초였으나"
- **권장**: "심장이 한 번 뛰는 시간 동안", "다음 숨을 쉬기 전에", "세상이 멈춘 것 같았다"
- 주관/객관 시간 대비를 쓸 때는 수치 없이 체감만으로: "체감으로는 오랜 순간이었다" (앞에 실제 시간 수치 붙이지 않는다)

## Dialogue Rules

### 형식
- 큰따옴표 사용: "대화 내용"
- 행동 비트와 함께 사용
- 화자 태그 최소화

### 시대적 대화 스타일
- **고어체 기반**: "~하시오", "~이로다", "~하겠소" (현대적 가독성 유지)
- **주인공 내면**: 현대어 ("이건 바둑으로 치면 패싸움이야")
- **주인공 대사**: 초반 어색 → 후반 자연스러운 고어체
- **무장**: 호탕하고 직설적 ("하하! 좋소!")
- **모사**: 우회적 ("주공, 한 수 말씀 올려도 되겠습니까")
- **군주**: 위엄과 인간미 ("과인이 직접 나서겠다")

### 대화 비율 가이드라인
- 대화: ~35%
- 행동/묘사: ~40%
- 내면 묘사: ~25%

## Word Choice

### 선호
- 구체적이고 감각적인 동사
- 바둑 용어의 자연스러운 전략 전환
- 시대에 맞는 명사 (관직명, 무기명, 지명)

### 회피
- 현대 속어/인터넷 밈 (주인공 내면 제외)
- 진부한 무협 클리셰 ("하늘을 가르는 검기")
- 과도한 한자어 나열
- "매우", "정말" 등 약한 수식어
- **"BCI 뇌가 자동으로 분석하는"** 유형 — 삼국시대에서 BCI는 작동하지 않는다. 세계관 위반.
- **충격 묘사 클리셰**: "눈 앞에 번개가 쳤다", "귓속에서 소리가 울렸다" 등 과부하/충격의 진부한 감각 표현. 이미지 연쇄를 살린 구체적 묘사로 대체.

### 전투씬 서술 원칙 (Battle Scene Mandatory Rules)

#### 핵심 규칙 (절대 준수)
- **모든 전투씬 상세 서술 의무**: "전투가 벌어졌다", "싸움이 끝났다" 식의 요약 서술 금지. 반드시 구체적 교환(exchange)으로 전개한다
- **주인공 개인 전투 필수 분량**: 서홍이 직접 참여하는 모든 전투에서 그의 개인 전투 장면이 해당 전투씬의 20% 이상을 차지해야 한다
- **미세 먼치킨(微 munchkin) 허용**: 대중성을 위해 주인공이 동급 이하 상대를 인상적으로 처리하는 순간을 포함한다. 단, 기(氣) 단계 절대 한계는 반드시 준수

#### 살상 묘사 원칙 (Lethal Combat — 필수 적용)
- **전투 살상은 현실적으로 묘사한다**: 전쟁터에서 검·창·화살은 사람을 죽인다. 이를 회피하거나 축소하면 전투의 무게와 긴장감이 사라진다. **살상 묘사는 금지되지 않는다.**
- **회피 금지 표현들**: 아래 표현들을 단독으로 쓰는 것 금지. 반드시 구체적 결과(상처·쓰러짐·사망)가 이어져야 한다
  - "쓰러뜨렸다" → "검이 옆구리를 꿰뚫었다. 쓰러졌다."
  - "무력화했다" → "어깨에 화살이 꽂혔다. 창을 놓쳤다."
  - "물러나게 했다" → 상처가 있는가? 이유가 있는가?
- **묘사 기준**: 삼국지연의·무협지 수준 — 검이 베면 피가 나고, 창이 꽂히면 관통되며, 치명상은 쓰러짐과 죽음으로 이어진다. 단, 진부한 고어(gore)의 반복 나열(피가 솟구쳤다×5) 금지. 결과를 보여주되 고통 포르노는 피한다
- **서홍의 검날 사용 원칙**: 검을 뽑으면 날을 사용한다. 자루(pommel) 타격은 포로를 원하거나 치명상을 피해야 하는 특수 상황에서만. "검 자루로 쳤다"가 기본 수단이 되어서는 안 된다
- **전투의 무게 의무**: 아군/적군 사상자를 구체적으로 묘사한다. 전투 후 빈 자리, 이름 없는 죽음들이 독자에게 전쟁의 실질적 비용을 느끼게 한다

#### 최소 서술 기준
| 대상 | 최소 교환 묘사 | 내면 독백 |
|------|-------------|---------|
| 서홍 직접 전투 (주력 장수급) | 5교환 이상 | 필수 — 판단 근거 포함 |
| 서홍 직접 전투 (잡병급) | 3교환 이상 + 명확한 결과 | 선택 |
| 부하 장수 전투 (왕대 등) | 3교환 이상 | 선택 |
| 집단 전투 묘사 | 핵심 장면 3씬 이상 | 선택 |

#### 서홍(서진우) 전투 스타일
- **수읽기 전투**: 상대의 다음 동작을 1-2수 미리 읽는 바둑적 감각. 마치 상대보다 한 박자 앞에 있는 것처럼 묘사. 실제로는 예측 능력이지 초인적 반사가 아님
- **최소 효율 원칙**: 화려한 기법 없이 최소 동작으로 최대 효과. 막기보다 빗나가게, 무너트리기보다 허를 찌름
- **기(氣) 활용 특성**: 기를 공격력에 쏟는 것이 아니라 신체 반응속도·균형·위치 선점에 활용. 자연스럽게 발현됨
- **약점 필수 유지**: 순수 무력 대결에서 발기(發氣) 이상 고수에게는 밀린다. 반드시 전술로 보완

#### 먼치킨 허용/금지 범위
- **허용**: 잡병을 날카롭게 처리(살상 포함), 동급 장수의 패턴을 읽어 3-5교환 내 결론, 위기에서 수읽기로 반전, 직접 싸우지 않고 전술로 강자 무력화
- **금지**: 기(氣) 단계 초과 무력, 홀로 수십 명 동시 처리, 갑작스러운 각성으로 한계 초월, 현실감 없는 무상(無傷) 클리셰 연속

#### 대인 전투 원칙 (1:1 Combat — 장수급 대결)
*(Ch.034 v5 리터칭에서 도출 — 2026-02-25)*

- **이름 부여 의무**: 서홍이 직접 맞서는 장수급 이상 적에게는 반드시 이름(성명 또는 직함+이름)을 부여한다. 이름 없는 익명 장수와의 결투는 전투의 무게가 절반으로 줄어든다
- **이름 교환 관례**: 장수급 1:1 대결 직전, 짧은 이름 교환 대사를 넣는다. 삼국지연의 전통 — "나는 ~이다" / "서홍이오". 2-3행으로 충분. 전투 전 결기의 순간이자 독자에게 상대의 존재를 각인시킨다
- **결말 원칙 — 결정적 타격**: 장수급 대결의 결말은 제압(무릎 꿇기)보다 치명적 타격(사망, 치명상)이 원칙이다. "무릎을 꿇었다"로 끝나는 결말은 카타르시스를 절반으로 줄인다. 예외: 포로로 쓸 필요가 있거나, 생존이 복선일 경우
- **위력 과시 반응 의무**: 장수급 적 처치 직후, 반드시 주변 병사들의 반응을 1-2문장 삽입한다. "물러나며 침을 삼켰다", "발이 묶였다", "흔들렸다" 등. 이는 단순 묘사가 아니라 AT1 업그레이드와 AT4 영향력의 Show Don't Tell 수단이다

#### 집단 전투 원칙 (Mass Battle — 전술 논리 서술)
*(Ch.034 v5 리터칭에서 도출 — 2026-02-25)*

- **승리 논리 명시 의무**: 수적으로 불리한 쪽이 이겼을 때, 독자가 납득할 수 있는 전술적 이유를 1-2문장으로 명시한다. "수적 열세지만 ~때문에 각개 전투에서 우세" 형식. 설명 없는 역전은 먹물주인공 클리셰로 전락한다
- **포위 효과 서술**: 포위가 완성된 순간, 그것이 전투력 배율에 미치는 영향을 한 번은 서술한다. 독자가 왜 이기는지 머릿속에서 계산할 수 있어야 카타르시스가 생긴다
- **개인전-집단전 교차 편집**: 집단전 묘사와 개인(서홍 또는 부하 장수) 전투 묘사를 교차 배치한다. 개인전의 결과가 집단전의 분위기를 바꾸는 카탈리스트 역할을 하도록 구성한다
- **충돌 감각 다양화**: 금속과 금속이 부딪힐 때 불똥, 갑옷 긁힘 소리, 충격이 팔을 타고 오는 감촉 등 시각 외 감각을 병행한다. 전투씬의 시각 편중은 몰입을 평면화한다

#### 바둑 기풍 전투 시스템 (Go Style Battle Framework)

서홍(서진우)의 전략 판단은 기억 속 프로 바둑 기사들의 기풍(棋風)을 전장에 대입한다.
전투 규모가 커질수록 개인 수읽기에서 부대 운용 전략으로 확장된다.

##### 4대 기풍 유형

| 기풍 | 기사 모델 | 핵심 원리 | 적합 전투 상황 |
|------|---------|---------|--------------|
| **침투·돌파형** | 이세돌(李世乭) | 급소 포착, 복잡함 유발, 역전 추구 | 공격전, 포위 돌파, 적 분리 |
| **견고·소모형** | 이창호(李昌鎬) | 완벽 방어, 최소 손실, 끝내기 승부 | 방어전, 지구전, 험지 수비 |
| **세력·전격형** | 조훈현(曺薰鉉) | 신속 세력 구축, 결정적 순간 전격 타격 | 속공전, 요충지 선점전 |
| **난전·백병형** | 유창혁/서봉수 | 복잡한 전투 유도, 아군 무력 우세 활용 | 혼전, 백병전, 근접전 |

##### 기풍별 서술 원칙

**이세돌형 — 침투·돌파형 (攻擊型)**
바둑 기풍: 상대 진영의 급소를 직관적으로 찾아내어 예상 밖 방향에서 침투. 복잡한 전투를 의도적으로 만들어 혼란 속 역전을 노림. 불리해도 포기하지 않는 역발상.
전투 적용:
- 적 방어선의 약점(급소)을 분석 → 집중 돌파, 병력 분리 후 각개격파
- 혼전을 의도적으로 유발 → 적 지휘 체계 붕괴
- 불리한 상황에서 역발상: 더 깊이 파고들어 역전의 한 수
필수 서술 3요소:
1. 급소 포착 내면 독백 ("저기다. 저 간격이 급소였다.")
2. 침투 후 적 반응 혼란 묘사 (대형 붕괴, 지휘 체계 갈팡질팡)
3. 혼전이 오히려 기회로 전환되는 역전 순간

**이창호형 — 견고·소모형 (守備型)**
바둑 기풍: 실수 없는 완벽한 방어로 상대의 자충수를 기다림. 끝내기(ヨセ)에서 작은 차이로 승리. "인용(忍用)" — 참고 기다리는 방식.
전투 적용:
- 지형(협로·고지·강변)을 방패로 견고한 방어선 구축
- 아군 피해 최소화 — 밀려도 대형은 유지
- 적의 과욕(자충수) 유도 → 무너지는 순간을 기다림
- 결정적 끝내기 시기: 적이 지쳤을 때 전면 공세
필수 서술 3요소:
1. 인내의 내면 독백 ("지금은 아니다. 저들이 먼저 지친다.")
2. 적의 과욕 → 자충수 순간 포착 (무리한 공세의 균열)
3. 끝내기 결단 순간 + 전면 일격

**조훈현형 — 세력·전격형 (速攻型)**
바둑 기풍: 초반부터 넓은 세력을 빠르게 구축하여 상대가 대응하기 전에 기정사실 만들기. 결정적 순간 전면 타격. 속도와 시기가 전략의 핵심.
전투 적용:
- 전장 초반 요충지(고지·교차로·강변) 신속 선점 → 판을 먼저 쥠
- 전선을 넓게 형성 → 적이 어디를 막아야 할지 혼란
- "이 순간이다" — 결단의 시기가 전투를 가른다
- 속도로 적의 대응을 무효화 (적이 배치 완료 전에 기정사실)
필수 서술 3요소:
1. 요충지 분석 내면 독백 ("저 언덕을 먼저 잡는 쪽이 판을 쥔다.")
2. 전격 이동의 속도감 (문장 흐름도 빠르고 짧게)
3. 광역 전선 완성 후 중심부 타격 결단

**유창혁/서봉수형 — 난전·백병형 (亂戰型)**
바둑 기풍: 복잡하고 어지러운 전투를 즐기며 타고난 싸움 감각으로 승리. 정석보다 실전 직관. 상대가 정리하려 할수록 더 뒤섞는다.
전투 적용:
- 아군 무력(왕대·태사자 등) 우세를 활용한 근접 혼전 유도
- 대형 붕괴 후 혼전 → 개인기 우위로 전세 역전
- 지휘관(서홍)보다 현장 장수들이 주도하는 구조
- 혼전으로 적 지휘 체계 무력화 (명령이 전달되지 않는 상황 만들기)
필수 서술 3요소:
1. 혼전 유도 의도 내면 독백 ("뒤섞이면 저들의 수가 의미 없어진다.")
2. 개인전-집단전 교차 편집 (장수들의 활약이 전세를 바꿈)
3. 혼전 결과: 아군 무력 우위로 최종 역전

##### 전투 규모별 기풍 적용 기준

| 규모 | 병력 기준 | 기풍 적용 방식 |
|------|---------|--------------|
| **소규모** | ~200명 이하 | 기풍 명시 불필요. 수읽기 개인 전투 중심. 기풍 요소 1-2개 자연스럽게 녹임 |
| **중규모** | 200~2,000명 | 전투 전 기풍 선언 내면 독백 의무. 병종 배치에 바둑 돌 개념 대응. 기풍 전환 가능(이유 명시 필수) |
| **대규모** | 2,000명 이상 | **기풍 선언 + 부대 포석 계획** 씬 1 전체 서술 필수. 적장 기풍 분석 포함. 기풍 충돌 구도 설정. 기풍 전환 = 전세 전환의 서사적 전환점 |

##### 대규모 전투 — 부대 배치 ↔ 바둑 돌 배치 매핑

| 바둑 개념 | 부대 운용 | 서술 예시 |
|----------|---------|---------|
| 화점(花點) 선점 | 요충지(고지·교차로) 선점 | "저 언덕이 화점이었다. 먼저 닿는 쪽이 판을 쥔다." |
| 연결(連結) | 부대 간 연락 유지 | "돌이 끊기면 죽는다. 연락이 끊긴 부대는 돌이 아니다." |
| 두터움(厚み) | 후방 지원 / 예비대 | "전면이 무너져도 뒤에서 받쳐야 한다." |
| 얇음(薄い) | 전선 과도 확장 경고 | "너무 넓으면 어디서든 무너진다." |
| 포위(包圍) | 적 부대 포위 섬멸 | "기(氣)를 막으면 돌은 죽는다. 퇴로를 먼저." |
| 사활(死活) | 포위된 아군 구출 여부 | "살릴 수 있는 돌인가. 아니면 버리고 다른 곳을 얻는가." |
| 축(シチョウ) | 도주로 차단 추격전 | "도망치는 방향에 이미 벽이 있었다." |
| 패(コウ) | 교착 / 소모전 | "이기려면 더 큰 것을 걸어야 한다." |
| 사석(捨石) | 의도적 부대 희생 | "이 돌 하나로 저 돌 셋을 살린다." |

##### 기풍 선언 내면 독백 형식 (중규모 이상 전투 필수)

[전황 파악] → [기풍 선택 이유] → [기풍 선언(암묵적)] → [구체적 운용 계획]

예시:
"지형이 협소했다. 병력을 넓게 펼 수 없었다.
이창호의 바둑이 떠올랐다 — 지키면서 이기는 바둑.
지금은 침투의 수가 아니었다.
협로를 방패로 쓰고, 저들이 지쳐 무너지길 기다려야 했다.
끝내기는 그때다."

##### 기풍 전환 원칙
전투 중 상황 변화로 기풍을 전환할 때:
- 전환 이유 내면 독백으로 명시 (기존 기풍이 왜 통하지 않는지)
- 새로운 상황 인식 → 기풍 재선언 → 즉각적 행동 변화 묘사
금지: 이유 없는 전환. 독자가 "왜 바뀌었나" 의문을 갖지 않아야 함.

### 바둑 용어 ↔ 전략 용어 매핑
- **포석(布石)**: 세력 배치, 첩자 배치, 거점 확보
- **정석(定石)**: 외교 관례, 표준 전술
- **수읽기(読み)**: 적의 다음 행동 예측
- **대국관(大局觀)**: 천하 대세 파악
- **사활(死活)**: 포위된 군의 생사 판단
- **끝내기(ヨセ)**: 영토 확정, 전쟁 마무리
- **패(コウ)**: 교착 상태, 반복되는 갈등
- **축(シチョウ)**: 추격전, 도주
- **집(地)**: 영토, 세력권
- **두터움(厚み)**: 견고한 방어, 후방 안정

### 문장 길이
- **전투**: 긴장감 있는 중문 위주. 극적 전환점에서만 단문 허용.
- **전략/사색**: 길고 복합적. 바둑 사고의 다층성 반영. 판단 근거와 고뇌를 충분히 서술.
- **감각 묘사**: 중~장문. 감각이 인물에게 주는 의미까지 포함하여 서술.
- **대화**: 짧은 대화 + 행동/내면 비트.
- **바둑 대국**: 장문 서술. 수의 의미, 형세 변화, 고뇌를 소설적으로 풀어쓰기. 용어는 독자용 설명을 자연스럽게 녹인다.

### 참고 작가/작품
- **나관중**: 삼국지연의 — 서사적 무게와 인물 조형
- **호다 요시에**: 히카루의 바둑 — 바둑 대국의 긴장감 묘사
- **달빛조각사(남희성)**: 성장물의 리듬감, 레벨업의 쾌감
- **전지적 독자 시점(싱숑)**: 회귀/시스템물의 구조, 독자 흡인력
- **견룡재야(보랏빛 하이에나)**: 삼국지 기반 판타지의 시대 분위기
- **하진(河晋)**: 전쟁터(War Trash) — 전장의 인간적 묘사

### 유머 가이드라인
- **주요 유머**: 현대인의 시대착오적 반응 (컵라면이 그리운 순간, "인터넷 검색하고 싶다")
- **바둑 오해**: 바둑 용어를 전쟁에 쓸 때 주변의 혼란
- **문화 충돌**: 현대적 사고 vs 삼국시대 관습
- **밀도**: 일상 장면에서만. 전투/비극 장면에서는 절대 금지.
