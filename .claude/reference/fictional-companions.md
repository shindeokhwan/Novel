# Fictional Companions - 가상 캐릭터 설계

초반(Part 1-2) 가상 동료 캐릭터 2-3명의 프로필, 역할, 희생 아크를 설계한다.

---

## 설계 원칙

### 가상 동료의 서사적 기능
1. **주인공 보호자**: Part 1에서 어리바리한 주인공을 보호/도와주는 존재
2. **감정 앵커**: 독자가 주인공의 세계에 정착하는 감정적 기반
3. **희생 카타르시스**: 일부 캐릭터의 희생 → 주인공 각성의 계기 (E5 대형)
4. **성장 거울**: 주인공의 변화를 비추는 거울 역할

### 규칙
- 가상 캐릭터는 **역사적 인물이 아닌 창작 인물**
- Part 1 Ch.1-5 사이 등장 (주인공과 유대 형성 충분한 시간 확보)
- 최소 1명은 **Part 2까지 생존** (장기 동료)
- 최소 1명은 **Part 1-2에서 희생** (각성 촉매)
- 희생 캐릭터는 최소 5챕터 이상 등장 후 희생 (감정 축적 필수)

## 가상 캐릭터 프로필

### 1. 이만(李萬) — "형 같은 사나이"

| 항목 | 내용 |
|------|------|
| 이름 | 이만(李萬) |
| 나이 | 30대 초반 |
| 출신 | 기주(冀州) 농민 출신, 황건적에 끌려옴 |
| 유형 | 파워형 무장. 덩치 크고 힘이 셈 |
| 무예 | 삼류(감기 전조). 기술 없이 힘으로 밀어붙이는 타입 |
| 성격 | 순박하고 충직. 말수 적고 행동으로 보여줌. 약한 자 괴롭히는 것 싫어함 |
| 첫 등장 | Ch.1-2 (황건적 동료, 주인공을 처음 도와주는 인물) |
| 역할 | Part 1 주인공 보호자. 몸이 적응 안 되는 서진우 대신 싸워줌 |
| 시그니처 | 말없이 먹을 것을 건네는 습관. "먹어." 한마디 |
| 운명 | **Part 2 전반 희생** (Ch.25-35 사이). 주인공을 지키다 전사 |

**서사적 기능**:
- 주인공이 처음 신뢰하는 사람 → 희생 시 감정 폭발(E5)
- "이만이 살아있었으면" — 이후 결정적 순간마다 떠오르는 이름
- 희생이 주인공의 "더 이상 잃지 않겠다" 결의의 기원

**희생 아크**:
1. Ch.1-2: 주인공을 도와줌 (보호)
2. Ch.3-10: 함께 성장, 전투, 일상 (유대 축적)
3. Ch.15-25: 부하로서 활약 (전투 동료)
4. Ch.25-35: **위기 상황에서 주인공 대신 부상/전사** → E5 대형
5. 이후: 주인공 내면에서 지속적으로 회상 (잔향)

### 2. 소영(小英) — "여동생 같은 군의(軍醫)"

| 항목 | 내용 |
|------|------|
| 이름 | 소영(蘇英) |
| 나이 | 10대 후반~20대 초반 |
| 출신 | 의사 가문 출신, 전란으로 가족 잃음 |
| 유형 | 비전투원. 약초/의술에 밝음 |
| 무예 | 없음 |
| 성격 | 밝고 수다스러움. 주인공의 까칠함에 개의치 않음. "다쳤으면 치료받으세요, 투덜거리지 말고." |
| 첫 등장 | Ch.3-5 (하곡촌 합류, 부상병 치료) |
| 역할 | 일상/온기 담당. 전투 사이 따뜻한 숨구멍. 화타 복선 연결 |
| 시그니처 | 약초 냄새. 손에 항상 묻어 있는 약재 |
| 운명 | **Part 2까지 생존**. 화타 합류 후 제자로 편입. 장기 조연 |

**서사적 기능**:
- 전투 사이 따뜻한 일상 씬 담당
- 주인공의 감정 변화를 가장 먼저 눈치채는 인물
- 화타 영입 복선의 연결고리
- 주인공이 "보호해야 할 사람"의 상징

### 3. 곽철(霍鐵) — "입 거친 검객"

| 항목 | 내용 |
|------|------|
| 이름 | 곽철(霍鐵) |
| 나이 | 20대 중반 |
| 출신 | 유랑 검객. 출신 불명. 본인도 부모를 모름 |
| 유형 | 민첩형 무장. 빠르고 날렵 |
| 무예 | 삼류~이류 경계. 재능 있으나 체계적 수련 부족 |
| 성격 | 입이 거칠고 불량. 권위를 인정 안 함. 그러나 은혜를 갚는 성격 |
| 첫 등장 | Ch.4-6 (하곡촌 근처에서 합류) |
| 역할 | 주인공과 티격태격하는 동료. 까칠한 주인공에게 까칠하게 대하는 유일한 인물 |
| 시그니처 | "형이라고 부르지 마, 졸병 나부랭이가." / 전투 후 항상 검에 묻은 피를 닦는 습관 |
| 운명 | **Part 1 후반~Part 2 초반 분기** — 아래 두 경로 중 택1 |

**운명 경로 A: 희생**
- Ch.13-14 안평현 공방전에서 주인공을 지키다 전사
- 죽기 직전 "...졸병 나부랭이가 현령이 됐네." 한마디
- 주인공 첫 전우의 죽음 → 전투의 무게 각인

**운명 경로 B: 이탈 후 재합류**
- Part 1 후반 자유를 찾아 떠남 (권위 거부)
- Part 2-3에서 성장하여 재등장 → 합류
- 주인공의 변화를 외부 시선으로 증언

## 희생 아크 체크리스트

가상 캐릭터 희생 시 아래 항목 점검:

- [ ] 최소 5챕터 이상 등장했는가? (감정 축적)
- [ ] 독자가 이 캐릭터를 좋아하게 만들었는가? (사소한 매력 포인트)
- [ ] 희생에 서사적 필연성이 있는가? (무의미한 죽음 금지)
- [ ] 주인공에게 감정적 충격을 주는가? (E5 대형)
- [ ] 희생 이후 주인공의 행동/결의 변화가 있는가?
- [ ] 이후 챕터에서 회상/잔향이 있는가? (최소 3회)
