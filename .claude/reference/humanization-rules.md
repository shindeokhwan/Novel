# Humanization Rules - AI 흔적 방지 규칙

AI 생성 흔적을 방지하고 인간적 필치를 유지하기 위한 작문 규칙.
`style-guide.md`를 보완하며, 모든 집필/수정 시 적용한다.

## 금지 패턴 (Forbidden Patterns)

### 절대 금지
- **시대착오 단어 (현대 외래어/영어 차용어)**: 삼국시대(2세기 말~3세기) 배경에서 대사·서술·내면 독백 막론하고 아래 단어 사용 절대 금지. AI 흔적 최상위 패턴.

  | 금지 단어 | 대체 표현 |
  |----------|----------|
  | 타이밍(timing) | 시기(時機) / 때 / 순간 / 시각(時刻) |
  | 리듬(rhythm) | 박자(拍子) / 흐름 |
  | 패턴(pattern) | 수순(手順) / 습관 / 결 / 움직임 |
  | 시스템(system) | 체계(體系) / 제도(制度) |
  | 스타일(style) | 방식(方式) / 방법(方法) |
  | 카리스마(charisma) | 위엄(威嚴) / 덕망(德望) / 풍모(風貌) |
  | 퍼센트(%) | 완전히 / 온전히 / 전부 |
  | 이미지(image) | 모습 / 형상(形象) |
  | 레벨(level) | 수준 / 단계 |
  | 에너지(energy) | 기력(氣力) / 힘 |
  | 포인트(point) | 요점 / 핵심 |

  **예외 없음**: 주인공 내면 독백도 동일하게 적용. 현대적 사고는 순한국어 또는 한자어로 표현.
  - **금지**: "타이밍이 중요했다" / "리듬이 잡혔소" / "같은 패턴이었다"
  - **허용**: "시기가 중요했다" / "박자가 잡혔소" / "같은 수순이었다"

- 수치적 시간 표현: "0.1초", "0.3초간", "찰나의 순간" → 체감적 표현 사용
- 동일 비유 3회 초과 반복: 같은 비유가 전체 원고에서 3회 넘으면 변주 필수
- 에피소드 종결 패턴 연속 2회 동일: 직전 에피소드와 같은 종결 방식 금지
- 감정 직접 수치화: "분노가 80%", "신뢰도가 올라갔다" 등
- **행동 묘사 SVO 단문 연쇄**: 행동 씬에서 "SVO. SVO. VO." 패턴이 3문장 이상 연속되는 것 절대 금지. AI 흔적 패턴 상위. `style-guide.md` SF01-SF05 참조.
  - **금지 예시**: `"말에서 내렸다. 창을 꺼냈다. 세 동작이 필요했다. 두 번 만에 했다."` → 4개 단문이 접속 없이 나열
  - **대안**: 동시 동작은 "~면서/~며" 통합, 연속 동작은 "~한 뒤/~자" 통합, 수치 동작은 인물 특성으로 대체, 설명 꼬리표는 삭제
- **챕터 번호 산문 참조**: "Ch.X에서", "X화 때" 등 → 시간/장소/사건의 구체적 서술로 대체 (style-guide.md 참조)
- **글자 수/수량 표현 오류**: 산문 내에서 "X 글자였다" 같은 표현을 쓸 때 실제 글자 수와 반드시 일치해야 함. 예: "수고했소"(4글자)를 "세 글자"로 쓰는 것 금지. 퇴고 시 반드시 직접 세어 확인.
- **`"[대사]." [이름]이/가 말했다.` 단독 패턴**: 수식어·행동 비트 없이 대사 직후 "[이름]이/가 말했다."로만 끝나는 형태 절대 금지. AI 흔적 패턴 최상위.
  - **금지 예시**: `"아직 모르오." 곽가가 말했다.` / `"기주에서 돌아왔다 했소." 서홍이 말했다.`
  - **대안 1 (태그 삭제)**: 화자가 이전 행동/내면으로 명확하면 태그 없이 대사만. `"아직 모르오. 쓰이지 않으면 알 수 없소."`
  - **대안 2 (행동 비트)**: `"아직 모르오." 잔을 탁자에 내려놓았다. "쓰이지 않으면..."`
  - **대안 3 (다른 동사)**: 물었다 / 답했다 / 이었다 / 덧붙였다 / 끊었다 / 받았다 — 말했다 금지
  - **예외**: "짧게 말했다", "돌아보지 않고 말했다" 등 수식어·행동이 붙은 형태는 허용
- **분리 호칭 대화 패턴**: `"[이름/호칭]." [인물]이 말했다. "[내용]"` — 같은 발화자의 대사를 이름 호칭과 내용으로 두 개의 따옴표로 쪼개는 구조 절대 금지. AI 흔적 패턴 1순위.
  - **금지 예시**: `"조우." 서홍이 말했다. "운수현 쪽 수계 자료 있으면 뽑아줘."` → 조우와 내용이 같은 발화인데 두 번 끊김
  - **대안 1 (합치기)**: `"조우, 운수현 쪽 수계 자료 있으면 뽑아줘." 서홍이 말했다.`
  - **대안 2 (행동 비트)**: `서홍이 조우를 보았다. "운수현 쪽 수계 자료 있으면 뽑아줘."`
  - **대안 3 (동작 전환)**: `서홍의 시선이 조우에게로 옮겨갔다. "운수현 쪽 수계 자료 있으면 뽑아줘."`
  - **예외**: 두 발화 사이에 상대방 반응이 있거나, 실제로 별개의 발화인 경우는 허용

### 주의 패턴 (3회 이상 시 경고)
- "입꼬리가 올라갔다/내려갔다" → 미소 표현 8종에서 분산
- "눈빛이 흔들렸다" → 눈 관련 표현 다양화
- "어깨가 축 처졌다" → 실망/피로 신체 반응 다양화
- "숨을 내쉬었다" → 호흡 표현 다양화
- "고개를 끄덕였다" → 동의 표현 다양화
- **"말했다" 챕터 내 누적**: 단독 "[이름]이/가 말했다."는 절대 금지(위 참조). 수식어 있는 형태도 챕터당 3회 초과 시 경고. 대안: 태그 삭제 / 행동 비트 / 답했다·받았다·끼어들었다·중얼거렸다·덧붙였다·이었다 등으로 다양화
- **em-dash(—) 남용**: 챕터당 5회 초과 시 경고
  - "A — B — C" 삽입어구 패턴 → 관형절 재구성 또는 쉼표로 대체
  - "~는 — 이것이었다" 반전 패턴 → 마침표+새문장으로 분리
  - "그때 —", "그리고 —" 접속사+em-dash → 단순 삭제
  - "A, B, C — 모두~" 열거 총괄 → 마침표+새문장 또는 쌍점
  - 같은 문단 내 em-dash 2회 이상 → 반드시 1회 이하로 축소

## 필수 다양성 규칙 (Diversity Requirements)

### 에피소드 종결 방식 (최소 5가지 순환)
1. **끊김형**: 행동 중단, 미완의 동작, 문장 미완성
2. **대화 잔향형**: 마지막 대사의 여운, 상대의 말이 귓전에 남는
3. **감각 단독형**: 하나의 감각만 부각 (소리만, 냄새만, 촉감만)
4. **질문형**: 미해결 질문으로 종결 (독자에게 던지는 것 아님, 캐릭터의 의문)
5. **타인 시선형**: POV 외 인물의 반응/시선으로 마무리

### 미소 표현 (최소 5가지)
- 입술 한쪽이 비틀어졌다 / 볼에 보조개가 팠다 / 이가 보일 만큼 웃었다
- 코끝이 찡그러지며 웃음이 났다 / 눈만 웃고 입은 일자였다
- 웃음이 새어나왔다 / 피식 소리가 났다 / 입을 손으로 가렸다

### 배경 감각 (씬당 3종 이상, 동일 조합 연속 금지)
- 시각: 깃발 펄럭임, 먼지 속 햇살, 검날 반사광, 빈 진영
- 청각: 전고(戰鼓) 소리, 말발굽, 검 부딪히는 금속음, 먼 함성, 바둑돌 놓는 소리
- 촉각: 검 자루의 땀, 갑옷 무게, 말 등의 체온, 바둑돌의 차가움
- 후각: 피 비린내, 솔밭 향, 밥 짓는 연기, 말 냄새, 먹 향
- 미각: 목구멍 건조함, 거친 밥의 거친 맛, 약초의 쓴맛, 군량의 퍽퍽함

## 대화 자연스러움 규칙 (Dialogue Authenticity)

### 금지
- 정보전달형 대사: 상대가 이미 아는 정보를 독자를 위해 설명하는 대사
- 완벽한 문장의 대화: 실제 대화는 문법적으로 불완전
- 감정 라벨링 대사: "나 지금 화났어", "나 슬퍼" (Show don't tell 위반)

### 필수 대화 노이즈 (챕터당 최소 3회)
1. **끊김**: "그러니까... 됐소, 직접 보여드리겠소."
2. **딴소리**: 대화 중간 관련 없는 행동 ("술이나 한 잔 하시오")
3. **비완성 문장**: "만약 이것이 진짜—" "진짜 뭐요?"
4. **오해**: 상대가 다른 의미로 받아들이기
5. **침묵**: 대답하지 않는 것으로 더 많은 것을 전달

### 캐릭터 음성 구분 규칙
- 대사만 읽어도 누구인지 알 수 있어야 함
- 서홍(서진우): 간결, 전략적 사고 섞인 발화, 바둑 비유, 초반 어색한 고어체→후반 자연스러움
- 왕대: 호탕, 직설적, 무인의 어투, 짧은 문장
- 전예: 과묵, 행동으로 말함, 짧고 확실한 대답
- 조우(모사): 우회적, 비유적, "주공" 호칭, 논리적 제안
- 만총(법관): 딱딱하고 정확한 어휘, 절제된 감정, 원칙 중시

## 갈등 깊이 규칙 (Conflict Depth)

### 갈등 지속 기간 최소 기준
- 소갈등 (개인 불화, 의견 차이): 최소 2 에피소드
- 중갈등 (팀 위기, 관계 균열): 최소 4 에피소드
- 대갈등 (구조적 위기, 배신): 최소 8 에피소드

### 해결 후 잔여 긴장 의무
- 화해 후에도 완전히 같지 않은 관계 묘사
- "해결됐지만 남은 것" — 미세한 어색함, 달라진 거리감
- 최소 2 에피소드 동안 잔여 긴장 유지

### 갈등 해결 금지 패턴
- 한 번의 대화로 완전 해소
- 모든 관계자가 동시에 납득
- 해결 후 즉시 이전보다 나은 관계

## 비합리성 쿼터 (Irrationality Quota)

### 챕터당 최소 1회
- 캐릭터의 비합리적 선택 (알면서도 잘못된 선택)
- 감정에 의한 비논리적 행동 (분노에 의한 실수, 집착)
- 자기기만 (자신의 진짜 감정을 외면)
- 회피 행동 (직면해야 할 것을 미루기)
- 과잉반응 (사소한 것에 과도하게 반응 — 다른 것의 투사)

### 인간적 비효율
- 불필요한 행동: 의미 없이 검집에서 검을 뽑았다 넣기, 이미 확인한 지도를 다시 펼치기
- 반복 행동: 같은 보고를 두세 번 되묻기, 이미 닫은 서찰을 다시 열어보기
- 모순 행동: 피곤하다면서 잠 안 자기, 배고프다면서 안 먹기
