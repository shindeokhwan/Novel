## 집필 규칙 요약 (core-rules.md 자동주입됨 — 아래는 강조 사항)

2. 시대착오어 11종 절대 금지
3. SVO 단문 3연속 금지 → 접속어미로 통합
4. 분리호칭·"말했다" 단독 패턴 금지
5. em-dash 에피소드당 최대 2회

7. POV: 3인칭 제한, 씬 내 전환 금지
8. 에피소드 종결 = 아웃라인 지정 방식 준수
