# Villain System - 악역 시스템

악역 설계 템플릿, 아크별 악역 배정, 응징 체크리스트를 정의한다.

---

## 악역 아크 4단계 템플릿

### 1단계: 등장(위협) — 1-2챕터
- 악역의 존재와 위협을 독자에게 각인
- 악역의 동기/배경 암시 (순수 악 금지)
- 주인공 측이 위협을 인지

### 2단계: 에스컬레이션(피해) — 2-3챕터
- 주인공 또는 부하에게 **직접적 피해** (필수)
- 피해 유형: 병력 손실, 장수 부상, 영토 상실, 배신, 민간 피해
- 악역이 우위에 있는 구간 — 독자 분노 축적

### 3단계: 대결 — 1-2챕터
- 주인공 측과의 정면 대결
- 악역의 전략/능력이 충분히 발휘됨
- 단순 먹치킨 금지 — 대등하거나 열세에서 역전

### 4단계: 응징(카타르시스) — 1챕터
- 악역이 **처벌**받는 순간 = E9 쾌감 패턴
- 주인공이 직접 또는 주도적으로 처리 (타인 처리 금지)
- 응징의 강도 ≥ 피해의 강도 (독자 납득)
- 응징 후 잔여 영향 1-2ep 유지 (피해의 흔적)

## 악역 설계 요소

### 필수 항목
| 항목 | 설명 |
|------|------|
| 이름/직함 | 네임드 악역 (이름 없는 악역 금지) |
| 동기 | 납득 가능한 적대 이유 (야심, 원한, 의무, 이해충돌) |
| 능력 | 주인공에게 실질적 위협이 되는 능력 (무력, 권모술수, 세력) |
| 피해 대상 | 누구에게 어떤 피해를 가하는가 |
| 응징 방식 | 어떻게 처벌받는가 |
| 지속 기간 | 최소 5챕터 |

### 악역 유형
| 유형 | 정의 | 예시 |
|------|------|------|
| 외부 적장 | 적 세력의 지휘관 | 태수, 적장, 토벌군 |
| 내부 배신자 | 아군 내 배신·첩자 | 첩자, 불만 세력 |
| 구조적 악역 | 제도/권력 구조 자체 | 부패 관리, 호족 횡포 |
| 경쟁자 | 같은 목표를 추구하는 라이벌 | 인재 쟁탈 경쟁 |

## Part별 악역 배정

### Part 1 (Ch.1-15)
- **악역 1**: [신규 설계 필요] — 안평현 지역 호족 또는 관군 장교
- 피해: 주인공 측 가상 캐릭터 부상/희생
- 응징: Ch.13-14 안평현 공방전에서 처단

### Part 2 전반 (Ch.16-35)
- **악역 2**: [신규 설계 필요] — 영천태수 측 핵심 부하 또는 세력 방해자
- 피해: 영천 확장 저지, 아군 피해
- 응징: Ch.34 태수 본대 격파 과정에서 처리

### Part 2 후반 (Ch.36-65)
- **악역 3**: [신규 설계 필요] — 여남 진출 방해 세력
- 피해: Ch.45 부상 사건 관련 가능
- 응징: 여남 평정 과정에서 처리

### Part 3 전반 (Ch.66-85)
- **악역 4**: 내부 배신자 (Ch.76-85 배신 아크)
- 피해: 정보 유출, 내부 분열
- 응징: Ch.85 배신자 정체 공개 → Ch.86 숙청

### Part 3 후반 (Ch.86-110)
- **악역 5**: 조조 측 장수/모사 (관도대전~중원 대전)
- 피해: 군사적 압박, 영토 위협
- 응징: 대전 승리로 격퇴

## 응징 체크리스트

악역 아크 완료 시 아래 항목 점검:

- [ ] 악역이 네임드인가? (이름 있는가)
- [ ] 악역의 동기가 납득 가능한가?
- [ ] 주인공/부하에게 직접적 피해를 가했는가?
- [ ] 피해가 구체적인가? (추상적 위협이 아닌가)
- [ ] 악역 지속 기간이 5챕터 이상인가?
- [ ] 주인공이 주도적으로 응징했는가?
- [ ] 응징 강도 ≥ 피해 강도인가?
- [ ] E9 쾌감 패턴으로 기록했는가?
