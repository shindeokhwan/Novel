# 챕터 아웃라인: Ch.092 "대전 2(大戰)"

## 챕터 정보
- **Part**: Part 3 중반전
- **위치**: Ch.92 / 150
- **시간**: 208년, 가을 (Ch.91 직후)
- **주요 위치**: 예주 북부 전선 — 진류 평원
- **챕터 성격**: 돌파형 (직접 행동, 전투, 결단)
- **클라이맥스 위치**: 중간배치 (긴장→폭발→여진)
- **엔딩 방식**: 대화잔향형
- **비합리성 쿼터**: 이미 확인한 지도를 반복해서 펼침

## 씬 1 (에피소드 1) — "교착(膠着)"
- **POV**: 서홍
- **위치**: 진류 평원 전선 — 서홍 본진
- **시간**: 208년 초가을, 이른 아침
- **목표**: 교전 3일째. 하후돈 선봉과의 교착 상태 + 서홍의 전략 수정
- **갈등**: 정면 교전은 수적 열세. 조조 본대 접근 중. 시간이 없음
- 감각: 전장 먼지, 피 냄새, 북소리, 갑옷 철 소리, 새벽 이슬
- 긴장도: 8

### 비트
1. 교전 3일째 보고. 장합 선봉 3,000이 하후돈 선봉 8,000과 교착. 아군 전사 200, 적 전사 500
2. 서홍 지도 분석 (비합리성 — 이미 확인한 지도 반복 펼침). 이창호형 방어+교착 전략의 한계 인식
3. 가후 서한 도착: 조조 본대 10만 진류 접근 중. 3일 내 합류 예상
4. 서홍 결단: 교착을 깨는 수. 조훈현형(과감한 전환) 기풍으로 전환

## 씬 2 (에피소드 2) — "전환(轉換)"
- **POV**: 서홍
- **위치**: 진류 평원 → 측면 우회로
- **시간**: 208년 초가을, 낮~오후
- **목표**: 서홍의 대전략 발동. 바둑 전략 최대 적용 (AT2 중 업그레이드)
- **갈등**: 위험한 기동 — 장료 기동대 800기를 하후돈 배후로 우회. 성공하면 포위, 실패하면 분산
- 감각: 말 갈기, 먼지, 저녁 석양, 갑옷 끈 조이는 소리
- 긴장도: 9
- **능력 업그레이드**: AT2 중 I04 상위형 — 천하 규모 실전 수읽기. 조조·유비·손권 삼자를 동시에 읽으며 전투 설계

### 비트
1. 서홍의 전략 설명: 정면(장합 수비) + 측면(장료 기동대 우회) + 예비(태사자 대기). 이중 포위 시도
2. 천하 규모 수읽기: 이 전투의 결과가 형주 전선(유비+손권 vs 조조)에 미칠 영향까지 계산. 여기서 하후돈을 꺾으면 조조 측면이 노출 → 형주 공격 속도 둔화 → 적벽까지 시간 확보
3. AT2 중 업그레이드 장면: "한 판의 수가 아니었다. 세 판의 수를 동시에 읽고 있었다"
4. 장료 기동대 출발. 서홍은 본진에서 전체를 조감. 고순 진영영 좌익 배치

## 씬 3 (에피소드 3) — "포위(包圍)"
- **POV**: 서홍
- **위치**: 진류 평원 본진 → 전장
- **시간**: 208년 초가을, 석양
- **목표**: 장료 우회 성공 + 하후돈 부대 측면 타격 + 교전 반전
- **갈등**: 포위 성공이나 하후돈이 돌파 시도. 조조 본대 2일 거리. 시간 경쟁
- 감각: 전투 소음, 피+먼지+쇠 냄새 혼합, 석양 역광, 함성
- 긴장도: 9→7 (전투 반전 후 여진)
- **엔딩**: 대화잔향형 — 전투 후 장합과의 짧은 대화

### 비트
1. 장료 기동대 측면 타격 성공. 하후돈 부대 혼란
2. 태사자 예비대 투입. 하후돈 진형 붕괴 시작
3. 하후돈 돌파 시도 — 서홍 직접 전투 없음(이 챕터에서는 지휘관). 다음 Ch.93에서 직접 전투
4. 결과: 하후돈 부대 후퇴. 아군 피해도 큼 (전사 800, 부상 1,500). 완전한 승리가 아닌 반전
5. 대화잔향 엔딩: 장합이 서홍 앞에 서서 "아직 반이오." 서홍 고개 끄덕임. 잔향.
