# Ch.057 아웃라인 — 독(毒)

## 챕터 개요

| 항목 | 내용 |
|------|------|
| 제목 | 독(毒) |
| 시점 | 195년 초여름, 안성 결절 완료 직후 |
| 핵심 사건 | 평여 방향 이동 중 정체불명의 과객(가후)과 조우. 천재적 식견에 놀라고, 정체를 추론한 뒤 "위험하지만 필요하다"는 판단에 도달 |
| 핵심 변화 | 서홍의 판 위에 '통제 불가능한 돌'이 처음 등장. 평여 확장이라는 단순 계획에 인재 변수 추가 |
| 독자 경험 | "이 사람은 대체 누구지?" → 정체 발각의 쾌감 → "이 사람을 다룰 수 있을까?"의 불안 |
| 챕터 성격 | **전환형** (예상→파열→재계산) |
| 씬 클라이맥스 | **중간배치** (씬1 축적 → 씬2 폭발 → 씬3 여운) |
| 쾌감 패턴 | E7 미니(진실발각 — 가후 정체 추론) + E3 미니(능력발현 — 가후 천재성) |
| 엔딩 방식 | **타인시선형** (서홍 POV에서 가후의 외적 반응 관찰) |
| AT 업그레이드 | AT2 소 I03 — 가후라는 극도로 복잡한 인물의 진의를 부분적으로 독해 |
| 서사줄기 | 천하40% / 인간30% / 정체성30% |

### 제목 의도
"독(毒)" — 세 가지 의미의 중첩:
1. 가후는 독사 같은 인물 (삼국지에서 "독사(毒士)"로 불림)
2. 독은 약이 될 수도, 사람을 죽일 수도 있다 — "위험하지만 필요하다"
3. 서홍의 기맥 치료도 화타의 침이라는 '독'이 필요 (약과 독의 경계)

## 씬1: 길 위의 눈 (축적)

| 항목 | 내용 |
|------|------|
| POV | 서홍 |
| 장소 | 안성 → 평여 방향 관도(官道), 이동 중 |
| 시간 | 이른 아침, 안성 결절 완료 사흘 후 |
| 줄기 | 천하35% / 인간45% / 정체성20% |
| 구조 유형 | III (외부 탐색+팀 관계) |
| 대화 유형 | B (보고+실행) |
| 감정 리듬 | W (파동 — 안정→불안→안정) |
| 분량 목표 | ~3,300자 |

### 비트 구성

**비트 1 — 이동 시작** (안성→평여 방향)
- 서홍, 태사자, 전예, 주방, 병사 약 120명의 이동 대열
- 안성 결절 완료 후 평여 방향 탐색 겸 이동. 목표: 평여현 주변 지형·호족 파악
- 서홍 내면: 지도 위의 수읽기 — 평여→상채 선을 연결하면 여남군 북부가 열린다. "두 점을 잇는 것이 선이 되고, 선이 형(形)이 된다."
- 배경 감각: 초여름 먼지 냄새(후각), 말 등의 열기(촉각), 관도 양쪽 보리밭의 황금빛(시각)

**비트 2 — 태사자의 시선** (P7 균열 씨앗)
- 이동 중 소규모 도적 잔당 5-6명과 조우. 전예가 처리하려 하자 태사자가 먼저 나섬
- **태사자가 서홍을 본다**: 서홍이 검에 손을 대지 않는 것을 관찰. 이동 중 내내 서홍이 전투를 회피하는 것이 태사자에겐 자연스럽지 않음
- 태사자의 외적 반응(서홍 POV에서 관찰): 도적 처리 후 태사자가 서홍 쪽을 잠깐 보는 시선. 무언가를 확인하려는 듯한 눈길
- 서홍 내면: 태사자의 시선을 감지. "저 눈이 무엇을 보고 있는지 안다. 칼을 쓰지 않는 태수를." 기맥 한계를 들킬까 하는 미세한 불안
- Ch.46 "검 끝 안 쓰시오" 복선과 연결 — 태사자의 관찰이 축적되고 있음을 보여줌

**비트 3 — 주방과의 거리** (P7 잔여 긴장)
- 이동 중 주방이 서홍에게 평여 방면 호족 정보를 보고. 그러나 보고가 끝난 뒤 돌아서는 주방의 등이 이전과 미묘하게 다름
- Ch.55-56에서의 균열→심화 이후, 주방은 더 가까워진 것 같으면서도 한 발 물러선 느낌
- 서홍 내면: "결절이 사람의 이야기이기도 하다는 것을 알았다. 알았는데, 알았다고 해서 다음 수가 보이는 것은 아니었다."
- 주방 대사(짧게): "평여 이남에 양씨(楊氏)가 있소. 영향력이 넓소." — 건조한 보고. 감정 없는 어조가 오히려 잔여 긴장을 드러냄

**비트 4 — 화타 근황** (F046 힌트 진행)
- 두기가 전령을 가지고 옴: 화타가 안성 부근에 도착했다는 소식. 내일 저녁 만남 가능
- 서홍 내면: 기맥 치료 진행 상황. "석 달이라 했다. 한 달이 지났다. 남은 두 달. 일곱 할에서 열 할로 가는 길이 남아 있었다."
- Ch.58 발기 달성의 포석 — 화타 치료가 순조롭게 진행 중임을 보여줌
- 감각: 기맥 상태를 의식하는 순간, 손가락 끝의 미세한 저림(촉각). 운기가 칠 할에서 막히는 느낌

## 씬2: 과객(過客) (폭발 — 클라이맥스)

| 항목 | 내용 |
|------|------|
| POV | 서홍 |
| 장소 | 관도 변 주막(酒幕) |
| 시간 | 오후, 이동 중 휴식 |
| 줄기 | 천하50% / 인간25% / 정체성25% |
| 구조 유형 | V (전환 — 예상치 못한 조우) |
| 대화 유형 | D (지략 대결 — 수읽기 vs 수읽기) |
| 감정 리듬 | R (상승 — 경계→경이→야심) |
| 분량 목표 | ~3,500자 |

### 비트 구성

**비트 1 — 주막 도착**
- 평여 방향 이동 중, 관도 변 주막에서 휴식
- 태사자·전예가 경계, 주방은 현지 상인에게 정보 수집
- 서홍은 구석 자리에 앉아 지도를 펼침
- 배경 감각: 주막의 질 낮은 탁주 냄새(후각), 삐걱거리는 나무 의자(청각), 창문으로 들어오는 먼지 섞인 햇빛(시각)

**비트 2 — 정체불명의 과객**
- 주막 한 구석에 혼자 앉아 있는 사내. 40대 중반, 검소한 여행복. 검을 차지 않음
- 사내가 서홍의 지도를 보며 한마디: "평여를 보고 계시오?" — 평범한 질문 같지만, 서홍이 지도에서 정확히 평여를 보고 있었다는 것을 한 눈에 파악한 것
- 서홍의 경계 발동. 보통 여행자가 아님을 직감

**비트 3 — 대화 (가후의 천재성 발현, E3 미니)**
- 과객이 자연스럽게 천하 정세에 대한 대화를 시작. 표면적으로 평범한 여행자의 잡담
- 그러나 몇 마디 속에 놀라운 정보 밀도:
  - "이각과 곽사는 올해 안에 갈라지오." — 장안 정세의 정확한 예측
  - "조조는 연주를 평정한 뒤 서주로 눈을 돌릴 것이오." — 조조의 다음 수를 읽음
  - "여남의 태수가 영천에서 왔다는 이야기를 들었소. 예주를 보는 눈이 있는 모양이오." — 서홍 세력을 이미 알고 있음
- 서홍 내면: "이 사내의 수읽기는 나와 다르다. 나는 판 위의 돌을 읽는다. 이 사내는 판 자체를 읽는다." (E3 — 서홍이 가후의 수준을 체감하는 순간)
- 과객의 결론: "때를 아는 것이 첫째이고, 사람을 아는 것이 둘째이며, 때와 사람이 맞을 때 물러날 줄 아는 것이 셋째이오." — 가후의 처세 철학을 함축한 대사

**비트 4 — 정체 추론 (E7 미니 — 진실발각)**
- AT2 소 I03 업그레이드 순간: 서홍이 과객의 대화 패턴, 장안 정세 지식, "때를 안다"는 철학에서 정체를 역산
- 서홍 내면: 바둑적 역산 — "장안의 내부를 이 정밀도로 읽는 사람은 안에 있었던 사람이다. 이각·곽사의 참모급. 그런데 지금 여기에 있다. 떠났다는 뜻이다."
- 결정적 추론: "때를 아는 것이 첫째라 했다. 이각·곽사에게서 때를 읽고 떠난 사람. 곽가가 말했다 — '때를 아는 사람이 있느냐'고." (F044 복선 회수 — 곽가 Ch.30 대사와 연결)
- 서홍이 묻지 않는다. 대신 지도를 접으며: "먼 길을 오셨소." — 다 알고 있다는 뉘앙스
- 과객(가후)의 반응: 잠깐 멈춤. 그리고 — "첫째만 아는 줄 알았는데, 둘째도 아시는 모양이오." (서홍이 자신의 정체를 추론했음을 인정)

**비트 5 — "위험하지만 필요하다"**
- 서홍 내면: 가후라는 인물의 위험성 인식. "독사(毒士). 이각에게 장안을 열어준 사람. 이 사내가 한 수를 두면 천하가 움직인다. 그런데 이 사내에겐 주군이 없다."
- "필요하다"의 인식: "수읽기만으로는 판을 이길 수 없다. 판 자체를 읽는 눈이 필요하다. 내가 가진 것은 돌의 수읽기이고, 이 사내가 가진 것은 판의 수읽기이다."
- 서홍이 직접 영입을 제안하지 않는다 — 가후는 그런 방식에 움직이는 사람이 아님을 직감
- 대신: "평여 쪽으로 가시오?" "그럴 수도 있소." "그 방향에 볼 것이 있을 것이오." — 다음 만남의 여지만 남김
- 가후의 대사: "태수의 판은 보았소. 수는 아직 모르겠소." — 세력은 봤지만 서홍이 어떤 사람인지는 아직 판단 보류

## 씬3: 수가 아직 모르겠소 (여운)

| 항목 | 내용 |
|------|------|
| POV | 서홍 |
| 장소 | 주막 → 이동 재개 |
| 시간 | 늦은 오후 |
| 줄기 | 천하35% / 인간20% / 정체성45% |
| 구조 유형 | IV (내면 — 가후 분석 + 자기 대면) |
| 대화 유형 | E (내면 독백 중심, 대화 최소) |
| 감정 리듬 | W (파동 — 야심→불안→결의) |
| 분량 목표 | ~3,100자 |

### 비트 구성

**비트 1 — 가후 떠난 뒤**
- 가후가 먼저 주막을 나섬. 서홍은 자리에 남아 가후가 앉았던 자리를 봄
- 가후가 남긴 것: 탁주 잔에 남은 술이 한 모금도 줄지 않았다 — 마시지 않았다. 경계를 풀지 않은 것
- 서홍 내면: "나도 경계했고, 저 사내도 경계했다. 바둑에서 첫 수를 놓기 전에 상대의 기풍(棋風)을 읽는 시간. 그 시간이 끝나지 않은 것이었다."

**비트 2 — 위험의 무게** (정체성 줄기)
- 가후의 과거를 복기: 이각에게 장안 진입 전략을 준 것이 가후. 왕윤이 죽고, 장안이 불타고, 헌제가 떠돌게 된 것이 가후의 한 수 때문
- "한 수로 천하를 뒤집은 사람. 그 수의 결과가 피와 불이었다."
- 서홍의 딜레마: 이런 사람을 곁에 두는 것은 독을 품는 것. 그러나 이런 사람 없이 조조를 이길 수 있는가?
- 현대적 사고(내면): "바둑에서 가장 위험한 수는 상대의 급소에 놓는 내 돌이 아니라, 내 진영 안에서 방향을 바꿀 수 있는 돌이다."
- 비합리성 쿼터: 서홍이 가후의 위험성을 알면서도 "필요하다"는 판단에 기우는 것 — 야심이 경계를 이기는 순간

**비트 3 — 주방에게 묻다**
- 서홍이 주방에게 가후에 대해 물음 (직접 이름을 대지 않고): "이각·곽사의 참모 중에 이름이 알려진 자가 있었소?"
- 주방: "가후(賈詡), 자 문화(文和). 장안을 열어준 사람이오." — 짧고 건조한 답. 그리고 한 박자 후: "만나셨소?"
- 서홍이 답하지 않음 (침묵 = 대화 노이즈). 주방이 더 묻지 않음 — 두 사람 사이의 새로운 신뢰. 물어도 답하지 않을 것을 알고, 물을 필요가 없음을 아는 것
- 주방 잔여 긴장: 답하지 않는 서홍에 대해 주방이 미세하게 입을 다무는 동작 — 이전 같으면 추궁했을 것. 지금은 참는다. 참는 것이 신뢰인지 체념인지는 아직 모른다

**비트 4 — 타인시선형 엔딩**
- 이동 재개. 서홍이 말 위에서 평여 방향을 바라봄
- 관도 멀리, 가후가 걸어가는 뒷모습이 아직 보임
- 가후가 한 번 뒤를 돌아봄 — 서홍과 눈이 마주침
- 가후의 외적 반응(서홍 POV에서 관찰): 돌아본 뒤 아무 표정 없이 다시 앞을 봄. 그러나 걸음이 — 평여 방향이다
- 서홍 내면 마지막: "평여 방향. 같은 방향이었다. 우연인지 의도인지. 가후라는 사내의 수읽기에서 우연은 없을 것이었다."
- 끝 — 가후가 어디로 가는지, 왜 돌아봤는지, 다시 만날 것인지 모두 미결. 독자에게 남는 것은 가후의 돌아보는 눈과, "수는 아직 모르겠소"라는 말
